第1課 李さんは中国人です

基本きほん

1. 李さんは 中国人です。
1. 李りさんは 中国ちゅうごく人じんです。
1. 小李是中国人。
  • 李りさん: [名词 + 敬称]
    • 李り: [名词] 姓氏"李"。
    • さん: [敬称接尾辞] 接在人名后，表示尊敬，相当于"先生"、"女士"、"小姐"。性别中立，适用范围广。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是"李さん"。"关于李先生/女士..."。
    • 读作 わ (wa)。
  • 中国ちゅうごく人じん: [名词]
    • 中国ちゅうごく: [名词] 中国。
    • 人じん: [接尾辞] 接在国名或地名后，表示"...地方的人"。
  • です: [助动词]
    • 丁寧体 (ていねいたい - Polite Style) 的判断助动词。用于名词或形容动词（な形容詞）后，表示肯定判断或陈述事实。
    • 相当于汉语的"是..."。
    • 使整个句子变得礼貌。
句意: 陈述"李先生/女士"的国籍是"中国人"这一事实，语气礼貌。
2. 森さんは 学生では ありません。
2. 森もりさんは 学生がくせいでは ありません。
2. 森先生不是学生。
  • 森もりさん: [名词 + 敬称]
    • 森もり: [名词] 姓氏"森"。
    • さん: [敬称接尾辞]。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 学生がくせい: [名词] 学生。
  • では ありません: [固定搭配] です 的否定形式 (丁寧体否定)。
    • で: 判断助动词 だ (です的简体) 的连用形 (或格助词)。
    • は: [係助詞] 在否定句中加强语气或对比。
    • ありません: 补助动词 ある 的否定丁寧形，表示"没有"或"不是"。【相关语法点: 存在动词否定】
    • 整体含义: "不是..."。
    • 口语缩略: では 在口语中常缩略为 じゃ，即 じゃ ありません。
句意: 说明"森先生/女士"的身份"不是学生"，语气礼貌。
3. 林さんは 日本人ですか。
3. 林はやしさんは 日本にほん人じんですか。
3. 林先生是日本人吗？
  • 林はやしさん: [名词 + 敬称]
    • 林はやし: [名词] 姓氏"林"。
    • さん: [敬称接尾辞]。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 日本にほん人じん: [名词]
    • 日本にほん: [名词] 日本。
    • 人じん: [接尾辞] 表示"...人"。
  • ですか: [助动词 + 終助詞]
    • です: [助动词] 表示丁寧体判断。
    • か: [終助詞 (しゅうじょし) - Sentence Ending Particle] 接在句末，表示疑问。通常读升调。
    • 结构: 陈述句 + か 构成疑问句。在丁寧体中，通常是 〜です + か。
句意: 询问"林先生/女士"的国籍是否是"日本人"，语气礼貌。
4. 李さんは JC企画の 社員です。
4. 李りさんは JC企画きかくの 社員しゃいんです。
4. 小李是JC策划公司的职员。
  • 李りさん: [名词 + 敬称]
    • 李り: [名词] 姓氏"李"。
    • さん: [敬称接尾辞]。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • JC企画きかく (ジェーシーきかく): [专有名词] JC策划 (公司名)。企画 意为策划、规划。
  • の: [格助詞 (かくじょし)]
    • 表示所属、所有关系。连接两个名词，名词1 の 名词2 表示"名词1的(所属的/相关的)名词2"。
    • 此处表示"JC企画"这家公司的。
  • 社員しゃいん: [名词] 公司职员、员工。
  • です: [助动词] 表示丁寧体肯定判断。
句意: 说明"李先生/女士"是"JC策划公司"的"职员"，语气礼貌。

基本きほん会話かいわ

A 甲: わたしは 李です。小野さんですか。
A 甲: わたしは 李りです。小野おのさんですか。
A 甲: 我姓李。(您)是小野女士吗？
  • わたし: [代名词] 我。(男女通用，比较礼貌的自称)
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 李り: [名词] 姓氏"李"。(自称时通常不加 さん，显得更自然)
  • です: [助动词] 表示丁寧体肯定判断。
  • 小野おのさん: [名词 + 敬称] 小野先生/女士。
    • 小野おの: [名词] 姓氏"小野"。
    • さん: [敬称接尾辞]。
  • ですか: [助动词 + 終助詞]
    • です: [助动词] 表示丁寧体判断。
    • か: [終助詞] 表示疑问。
句意: 甲自我介绍姓李，并向乙确认对方是否是小野。
乙: はい、そうです。小野です。
乙: はい、そうです。小野おのです。
乙: 是的，(我)是小野。
  • はい: [感叹词/应答词] 是的，嗯。(表示肯定回答)
  • そうです: [固定表达] 是那样的。
    • そう: [副词/代词] 那样，如此。指代前文提到的内容或状态。
    • です: [助动词] 表示丁寧体肯定判断。
    • 用法: 用于肯定对方的陈述或疑问，表示"是的，你说得对/情况是这样的"。
  • 小野おの: [名词] 姓氏"小野"。(自称时通常不加 さん)
  • です: [助动词] 表示丁寧体肯定判断。
句意: 乙肯定甲的猜测("您是小野女士吗？" -> "是的")，并再次确认自己是小野。
B 甲: 森さんは 学生ですか。
B 甲: 森もりさんは 学生がくせいですか。
B 甲: 森先生是学生吗？
  • 森もりさん: [名词 + 敬称] 森先生/女士。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 学生がくせい: [名词] 学生。
  • ですか: [助动词 + 終助詞] 构成丁寧体疑问。
句意: 甲询问森先生/女士的身份是否是学生。
乙: いいえ、学生では ありません。会社員です。
乙: いいえ、学生がくせいでは ありません。会社かいしゃ員いんです。
乙: 不，不是学生。是公司职员。
  • いいえ: [感叹词/应答词] 不，不是。(表示否定回答)
  • 学生がくせい: [名词] 学生。
  • では ありません: [固定搭配] です 的丁寧体否定形式。"不是..."。
    • で + は + ありません。
    • 口语常缩略为 じゃ ありません。
  • 会社かいしゃ員いん: [名词] 公司职员。
    • 会社かいしゃ: [名词] 公司。
    • 員いん: [接尾辞] 表示"...成员"。
  • です: [助动词] 表示丁寧体肯定判断。
句意: 乙否定甲的猜测，说明自己不是学生，而是公司职员。
C 甲: 吉田さんですか。
C 甲: 吉田よしださんですか。
C 甲: 您是吉田先生吗？
  • 吉田よしださん: [名词 + 敬称] 吉田先生/女士。
  • ですか: [助动词 + 終助詞] 构成丁寧体疑问。
句意: 甲询问对方是否是吉田。
乙: いいえ、ちがいます。森です。
乙: いいえ、ちがいます。森もりです。
乙: 不，不是。我是森。
  • いいえ: [感叹词/应答词] 不。
  • ちがいます: [动词]
    • 原型: 違ちがう (五段/Group 1)。意为"不同"、"错误"、"不对"。
    • 形式: ます形 (丁寧体)。
    • 用法: 用于否定对方的猜测或说法，"不对"、"不是的"。
  • 森もり: [名词] 姓氏"森"。(自称)
  • です: [助动词] 表示丁寧体肯定判断。
句意: 乙否定甲的猜测("您是吉田吗？" -> "不，不对")，并告知自己姓森。
D 甲: 李さんは JC企画の 社員ですか。
D 甲: 李りさんは JC企画きかくの 社員しゃいんですか。
D 甲: 小李是JC策划公司的职员吗？
  • 李りさん: [名词 + 敬称]。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • JC企画きかく: [专有名词] JC策划公司。
  • の: [格助詞] 表示所属关系，"..."的。
  • 社員しゃいん: [名词] 公司职员。
  • ですか: [助动词 + 終助詞] 构成丁寧体疑问。
句意: 甲询问李先生/女士是否是JC策划公司的员工。
乙: はい、そうです。
乙: はい、そうです。
乙: 是的。
  • はい: [感叹词/应答词] 是的。
  • そうです: [固定表达] 是那样的。(肯定对方的疑问)
句意: 乙肯定甲的询问("小李是JC策划的职员吗？" -> "是的")。

  • はい: [感叹词/应答词] 是的。
  • 李り秀しゅう麗れい: [名词] 姓名"李秀丽"。(自称)

句意: 李确认自己身份，并进行初次见面的标准问候语。
小野: はじめまして、小野緑です。

  • です: [助动词] 表示丁寧体肯定判断。

句意: 李误以为森是吉田先生。

  • よろしく: [副词/寒暄语] (省略了 お願ねがいします 的简略说法，但也足够礼貌)
