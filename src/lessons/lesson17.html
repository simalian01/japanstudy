第17課 わたしは 新しい 洋服が 欲しいです

基本きほん

1. わたしは 新しい 洋服が 欲しいです。
1. わたしは 新あたらしい 洋服ようふくが 欲ほしいです。
1. 我想要(买)新西服。
  • わたし: [代名詞] 我。
  • は: [係助詞]
    • 提示主题: 表明句子谈论的对象是“わたし”。
    • 读作 わ (wa)。
  • 新あたらしい: [イ形容詞] 新的。
  • 洋服ようふく: [名詞] 西式服装，西装。
  • が: [格助詞]
    • 提示欲望的对象: 在 `～が 欲しい` 句型中，表示想要的东西。
  • 欲ほしい: [イ形容詞] 想要的。
    • 表示说话人想要得到某个具体物品时的愿望。
    • 通常与格助词 `が` 连用，形成 `[名词]が 欲しい` 的形式。
    • 只能用于表达第一人称的愿望，或在疑问句中询问第二人称的愿望。
  • です: [助動詞]
    • 丁寧体 (ていねいたい - Polite Style) 的结尾。用于形容词后，使句子更礼貌。
句意: (我)想要新的西服。表达说话人对新西服的愿望。
2. わたしは 映画を 見たいです。
2. わたしは 映画えいがを 見みたいです。
2. 我想看电影。
  • わたし: [代名詞] 我。
  • は: [係助詞]
    • 提示主题: 表明句子谈论的对象是“わたし”。
    • 读作 わ (wa)。
  • 映画えいが: [名詞] 电影。
  • を: [格助詞]
    • 提示宾语: 表示动作 `見ます` (看) 的直接对象。
  • 見みたい: [動詞 + 助動詞] 想看。
    • 原型: 見る (みる) [动词] 看。
    • 形式: 動詞ます形 (`見ます` -> `見`) + 助動詞 `たい`。
    • 含义: 表示说话人想要进行某个动作的愿望。
    • 只能用于表达第一人称的愿望，或在疑问句中询问第二人称的愿望。
    • 和 `欲しい` 不同，`たい` 表示的是做某事的愿望。
    • `たい` 本身活用类似イ形容詞。
  • です: [助動詞]
    • 丁寧体 (ていねいたい - Polite Style) 的结尾。附加在 `たい` 之后，使句子更礼貌。
句意: (我)想看电影。表达说话人想进行“看电影”这个动作的愿望。
3. いっしょに お茶を 飲みませんか。
3. いっしょに お茶ちゃを 飲のみませんか。
3. 一起喝茶好吗？
  • いっしょに: [副詞] 一起。
  • お茶ちゃ: [名詞] 茶。`お` 是美化语前缀。
  • を: [格助詞]
    • 提示宾语: 表示动作 `飲みます` (喝) 的直接对象。
  • 飲のみませんか: [動詞 + 助動詞 + 助詞] 不喝(吗)? / 一起喝...怎么样?
    • 原型: 飲む (のむ) [动词] 喝。
    • 形式: 動詞ます形 (`飲みます` -> `飲み`) + 否定助動詞 `ません` + 疑問助詞 `か`。
    • 含义: `～ませんか` 是提出邀请或建议的常用句型，比 `～ましょう` 语气更委婉。虽然形式上是否定疑问句，但实际上表示“要不要一起做...?”。
句意: 要不要一起喝茶？ (委婉地邀请对方一起喝茶)。
4. ちょっと 休みましょう。
4. ちょっと 休やすみましょう。
4. 休息一下吧！
  • ちょっと: [副詞] 稍微，一下。
  • 休やすみましょう: [動詞 + 助動詞] 休息吧。/ 让我们休息吧。
    • 原型: 休む (やすむ) [动词] 休息。
    • 形式: 動詞ます形 (`休みます` -> `休み`) + 意志/勧誘助動詞 `ましょう`。
    • 含义: `～ましょう` 是动词的意志形 (Volitional Form) 的礼貌体。主要用于：
      • 提议一起做某事 (Let's...): 如本句。
      • 主动提出为对方做某事 (Shall I...?): 例如，`持ちましょうか` (要不要我帮你拿?)。
句意: 让我们稍微休息一下吧。 (提议大家一起休息)。

基本きほん会話かいわ

A 甲: 今 何が いちばん 欲しいですか。
A 甲: 今いま 何なにが いちばん 欲ほしいですか。
A 甲：(你)现在最想要什么？
  • 今いま: [名詞/副詞] 现在。
  • 何なに: [代名詞] 什么。
  • が: [格助詞]
    • 提示欲望的对象: 与 `欲しい` 连用，询问想要的东西是什么。
  • いちばん: [副詞] 最，第一。表示程度最高。
  • 欲ほしい: [イ形容詞] 想要的。
  • です: [助動詞] 礼貌体结尾。
  • か: [終助詞] 疑问助词，表示问句。
句意: 现在最想要的是什么？
乙: 安い 車が 欲しいです。
乙: 安やすい 車くるまが 欲ほしいです。
乙：想要辆便宜点儿的车。
  • 安やすい: [イ形容詞] 便宜的。
  • 車くるま: [名詞] 车。
  • が: [格助詞]
    • 提示欲望的对象: 表示想要的东西是“便宜的车”。
  • 欲ほしい: [イ形容詞] 想要的。
  • です: [助動詞] 礼貌体结尾。
句意: (我)想要便宜的车。
B 甲: 今日 デパートへ 買い物に 行きます。李さんも いっしょに どうですか。
B 甲: 今日きょう デパートへ 買かい物ものに 行いきます。李りさんも いっしょに どうですか。
B 甲：今天(我)去百货商店买东西。小李一起去怎么样？
  • 今日きょう: [名詞] 今天。
  • デパート: [名詞] 百货商店 (Department Store)。
  • へ: [格助詞]
    • 提示移动方向: 表示动作“去”的目的地。这里也可以用 `に`。
  • 買かい物もの: [名詞] 购物。
  • に: [格助詞]
    • 提示移动目的: 和移动动词 (行く、来る、帰る 等) 连用，后续常接表示动作的名词或动词ます形，表示去做某事。这里表示“去购物”。
    • 结构: [地点] へ/に [目的(名词/动词ます形)] に 行く/来る/帰る。
  • 行いきます: [動詞] 去 (ます形，礼貌体)。原型是 `行く (いく)`。
  • 李りさん: [固有名詞 + 接尾詞] 小李/李先生/李女士。`さん` 是常用的敬称。
  • も: [係助詞]
    • 表示“也”: 暗示除了说话人自己去之外，询问小李是否也去。
  • いっしょに: [副詞] 一起。
  • どうですか: [表現] 怎么样？
    • 用来询问对方的意见或提议。比 `どう` 更礼貌。
句意: 今天我去百货商店购物。小李，(你)也一起去怎么样？
乙: はい、ぜひ 行きたいです。
乙: はい、ぜひ 行いきたいです。
乙：好，(我)一定去。
  • はい: [感動詞] 是，好的。表示肯定或应答。
  • ぜひ: [副詞] 一定，务必。强调强烈的愿望或请求。常与 `～たい` 或 `～てください` 连用。
  • 行いきたい: [動詞 + 助動詞] 想去。
    • 原型: 行く (いく) [动词] 去。
    • 形式: 動詞ます形 (`行きます` -> `行き`) + 助動詞 `たい`。
    • 含义: 表示想要进行“去”这个动作的愿望。
  • です: [助動詞] 礼貌体结尾。
句意: 好的，(我)非常想去。
C 甲: 李さん、何を 食べたいですか。
C 甲: 李りさん、何なにを 食たべたいですか。
C 甲：小李，(你)想吃什么？
  • 李りさん: [固有名詞 + 接尾詞] 小李。
  • 何なに: [代名詞] 什么。
  • を: [格助詞]
    • 提示宾语: 在 `～を 食べたい` 句型中，表示想要吃的动作的对象。虽然 `たい` 前面有时可以用 `が`，但在询问具体想吃/喝/做什么时，常用 `何を～たいですか`。
  • 食たべたい: [動詞 + 助動詞] 想吃。
    • 原型: 食べる (たべる) [动词] 吃。
    • 形式: 動詞ます形 (`食べます` -> `食べ`) + 助動詞 `たい`。
    • 含义: 表示想要进行“吃”这个动作的愿望。
  • です: [助動詞] 礼貌体结尾。
  • か: [終助詞] 疑问助词。
句意: 小李，你想吃什么？
乙: 何でも いいです。
乙: 何なんでも いいです。
乙：什么都行。
  • 何なんでも: [副詞] 什么都...，无论什么。
    • 由疑问词 `何 (なん)` + 助词 `で` + 助词 `も` 构成。`疑问词 + でも` 表示全面肯定或全面否定（与否定呼应时）。这里表示“无论什么都”。
    • 读作 `なんでも (nandemo)`。
  • いい: [イ形容詞] 好的，可以的。
  • です: [助動詞] 礼貌体结尾。
句意: 吃什么都可以。
D 甲: 土曜日の 午後、コンサートへ 行きませんか。
D 甲: 土曜日どようびの 午後ごご、コンサートへ 行いきませんか。
D 甲：星期六下午去听音乐会好吗？
  • 土曜日どようび: [名詞] 星期六。
  • の: [格助詞] 连接名词，表示所属或时间，“...的”。这里表示“星期六的”。
  • 午後ごご: [名詞] 下午，P.M.。
  • コンサート: [名詞] 音乐会 (Concert)。
  • へ: [格助詞]
    • 提示移动方向: 表示动作“去”的目的地。这里也可以用 `に`。
  • 行いきませんか: [動詞 + 助動詞 + 助詞] 不去(吗)? / 一起去...怎么样?
    • 原型: 行く (いく) [动词] 去。
    • 形式: 動詞ます形 (`行きます` -> `行き`) + 否定助動詞 `ません` + 疑問助詞 `か`。
    • 含义: `～ませんか` 是提出邀请或建议的常用句型，语气委婉。表示“要不要一起去...?”。
句意: 星期六下午，要不要一起去音乐会？ (发出邀请)。
乙: いいですね。行きましょう。
乙: いいですね。行いきましょう。
乙：好啊，去吧。
  • いい: [イ形容詞] 好的。
  • です: (省略) `いいですね` 是 `いいですね` 的口语省略形式，表示赞同。
  • ね: [終助詞]
    • 表示征求同意或确认: 表示“...呀/啊”，带有感叹或寻求共鸣的语气。
  • 行いきましょう: [動詞 + 助動詞] 去吧。/ 让我们去吧。
    • 原型: 行く (いく) [动词] 去。
    • 形式: 動詞ます形 (`行きます` -> `行き`) + 意志/勧誘助動詞 `ましょう`。
    • 含义: `～ましょう` 在这里表示接受提议，并表示“那我们一起去吧”。
句意: 好呀。(我们)一起去吧。 (表示同意邀请并提议一起行动)。

李: 小野さんは 何を お願いしましたか。
李: 小野おのさんは 何なにを お願ねがいしましたか。
李： 小野さん向神(佛)许了什么愿？
  • 小野おのさん: [固有名詞 + 接尾詞] 小野先生/女士。
  • は: [係助詞] 提示主题。
  • 何なに: [代名詞] 什么。
  • を: [格助詞] 提示宾语，表示动作 `お願いする` 的对象。
  • お願ねがいしましたか: [動詞 + 助動詞 + 助詞] 祈愿了(吗)? / 请求了(吗)?
    • 原型: お願いする (おねがいする) [动词] 拜托，请求，祈愿。(「お願い」是名词「願い」的美化语，「する」是动词)
    • 形式: `します` (ます形) -> `しました` (过去式ます形) + `か` (疑问助词)。
    • 含义: 询问对方祈求了什么。
句意: 小野先生/女士，您祈求了什么？
小野: 健康と 恋愛です。
小野: 健康けんこうと 恋愛れんあいです。
小野： 健康和恋爱。
  • 健康けんこう: [名詞] 健康。
  • と: [並列助詞] 和。用于并列名词。
  • 恋愛れんあい: [名詞] 恋爱。
  • です: [助動詞] (省略了主语和动词 `お願いしました`) 表示祈愿的内容是“健康和恋爱”。
句意: (我祈求了)健康和恋爱。
李: 恋愛ですか。
李: 恋愛れんあいですか。
李： 恋爱？
  • 恋愛れんあい: [名詞] 恋爱。
  • です: [助動詞] (这里起确认作用)。
  • か: [終助詞] 疑问助词，表示惊讶或确认。
句意: (是)恋爱吗？ (表示有点惊讶或想确认)。
小野: ええ。今年中に 結婚したいです。
小野: ええ。今年中ことしじゅうに 結婚けっこんしたいです。
小野： 嗯。想在今年内结婚。
  • ええ: [感動詞] 嗯，是的。比 `はい` 稍微口语化一些。
  • 今年中ことしじゅう: [名詞/副詞] 今年之内。`～中 (じゅう)` 接在时间名词后表示“在...期间内”。
  • に: [格助詞] 表示时间点或时间范围。这里指“在今年之内”。
  • 結婚けっこんしたい: [動詞 + 助動詞] 想结婚。
    • 原型: 結婚する (けっこんする) [动词] 结婚。
    • 形式: 動詞ます形 (`します` -> `し`) + 助動詞 `たい`。
    • 含义: 表示想要进行“结婚”这个动作的愿望。
  • です: [助動詞] 礼貌体结尾。
句意: 嗯。我想在今年内结婚。
李: 相手は いますか。
李: 相手あいては いますか。
李： 有对象吗？
  • 相手あいて: [名詞] 对象，伙伴。(在这里指结婚对象)。
  • は: [係助詞] 提示主题。
  • いますか: [動詞 + 助詞] 有(吗)?
    • 原型: いる [动词] 存在 (用于有生命或能自主移动的物体)。
    • 形式: ます形 (`います`) + 疑問助詞 `か`。
    • 含义: 询问是否存在。
句意: 有(结婚)对象吗？

  • が: [格助詞] 提示欲望的对象。
  • 欲ほしい: [イ形容詞] 想要的。

    • 原型: まじめだ [形容动词] 认真的。

  • 寒さむい: [イ形容詞] 冷的 (天气)。
  • です: (省略) 口语中形容词后常省略 `です`。
  • ね: [終助詞] 表示感叹或寻求共鸣，“...啊”。
  • 温あたたかい: [イ形容詞] 温暖的，热的 (指食物、饮料、气候等)。
  • 物もの: [名詞] 东西，事物。`温かい物` 指热的东西(通常指食物或饮料)。
  • を: [格助詞] 提示宾语。
  • 食たべませんか: [動詞 + 助動詞 + 助詞] 不吃(吗)? / 一起吃...怎么样?
    • 原型: 食べる (たべる) [动词] 吃。
    • 形式: 動詞ます形 (`食べます` -> `食べ`) + 否定助動詞 `ません` + 疑問助詞 `か`。
    • 含义: `～ませんか` 用于委婉地提议或邀请。
句意: 有点冷呢。要不要吃点热的东西？ (提议一起吃点热食)。
李: そうですね。
李: そうですね。
李： 是啊。
  • そう: [副詞] 那样。
  • です: [助動詞] 是。
  • ね: [終助詞] 表示同意或附和，“是啊”。
  • そうですね: [表現] 是这样呢。用于表示同意对方的说法或感受。
句意: 是啊。(同意小野觉得冷的说法)。
小野: 何が いいですか。
小野: 何なにが いいですか。
小野： 吃什么好呢？
  • 何なに: [代名詞] 什么。
  • が: [格助詞] 提示评价的对象。
  • いい: [イ形容詞] 好的。
  • です: [助動詞] 礼貌体结尾。
  • か: [終助詞] 疑问助词。
句意: 什么好呢？ (询问对方想吃什么热食)。
李: 何でも いいですよ。
李: 何なんでも いいですよ。
李： 什么都行。
  • 何なんでも: [副詞] 什么都...，无论什么。
  • いい: [イ形容詞] 好的，可以的。
  • です: (省略) 省略了 `です`。
  • よ: [終助詞] 用于告知对方其可能不知道的信息或强调自己的主张，带有亲切或确认的语气。这里表示“（我这边）吃什么都行哦”。
句意: 什么都行啊。
小野: じゃあ、お汁粉は どうですか。いい 店を 知っています。そこへ 行きましょう。
小野: じゃあ、お汁粉しるこは どうですか。いい 店みせを 知しっています。そこへ 行いきましょう。
小野： 那么，(吃)年糕小豆汤怎么样？我知道个好店。去那儿吧。
  • じゃあ: [接続詞] 那么。
  • お汁粉しるこ: [名詞] 年糕小豆汤。`お` 是美化语。
  • は: [係助詞] 提示主题，提出“年糕小豆汤”作为讨论/建议的对象。
  • どうですか: [表現] 怎么样？
  • いい: [イ形容詞] 好的。
  • 店みせ: [名詞] 店铺。
  • を: [格助詞] 提示宾语，表示 `知っています` 的对象是“好店”。
  • 知しっています: [動詞] 知道。
    • 原型: 知る (しる) [动词] 知道，了解。
    • 形式: `て形` (`知って`) + `います`。表示持续的状态，“知道”这个状态一直持续着。
  • そこ: [代名詞] 那里 (指代前面提到的“好店”的位置，通常离听话人较近或双方都知道的地方)。
  • へ: [格助詞] 提示移动方向。
  • 行いきましょう: [動詞 + 助動詞] 去吧。/ 让我们去吧。
    • 原型: 行く (いく) [动词] 去。
    • 形式: 動詞ます形 (`行きます` -> `行き`) + 意志/勧誘助動詞 `ましょう`。
    • 含义: 提议一起去。
句意: 那么，年糕小豆汤怎么样？我知道一家好店。我们去那里吧。
李: お汁粉？ ぜひ 食べたいです。
李: お汁粉しるこ？ ぜひ 食たべたいです。
李： 年糕小豆汤？一定得尝尝。
  • お汁粉しるこ？: [名詞] 年糕小豆汤？ (带有疑问或确认的语气)。
  • ぜひ: [副詞] 一定，务必。强调强烈的愿望。
  • 食たべたい: [動詞 + 助動詞] 想吃。
    • 原型: 食べる (たべる) [动词] 吃。
    • 形式: 動詞ます形 (`食べます` -> `食べ`) + 助動詞 `たい`。
    • 含义: 表示想要进行“吃”这个动作的愿望。
  • です: [助動詞] 礼貌体结尾。
句意: 年糕小豆汤？(我)非常想吃。
