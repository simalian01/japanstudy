第27課 子供の時、大きな地震がありました

基本きほん

1. 子供の時、大きな地震がありました。
1. 子供こどもの時とき、大おおきな地震じしんがありました。
1. （我）小时候，发生过大地震。
  • 子供こども: [名词] 孩子，儿童。
  • の: [格助詞]
    • 连接名词，表示所属或修饰关系。在这里，子供の時 意为“孩子的时候”，即“小时候”。
  • 時とき: [名词] 时候，时间。
  • 、: [読点] 逗号，表示停顿。
  • 大おおきな: [連体詞 (れんたいし)]
    • 修饰名词，表示“大的”。
    • 与形容词「大きい」意思相同，但「大きな」只能直接修饰名词，不能作谓语。
    • 形式: 形容動詞「〜だ」的連体形「〜な」的形式，但属于連体詞。
  • 地震じしん: [名词] 地震。
  • が: [格助詞]
    • 提示主语：表示存在的对象是“地震”。在「あります」「います」句型中，表示存在的对象。
  • ありました: [动词]
    • 原型: ある (aru)
    • 形式: 「ある」的ます形「あります」的过去式「ありました」。
    • 含义: 表示（无生命物体的）存在或发生。在这里指地震的发生。使用过去式表示是过去发生的事情。
    • 属于五段动词。
句意: 叙述过去（小时候）曾经发生过一次大地震。
2. 映画を見る時、いつもいちばん後ろの席に座ります。
2. 映画えいがを見みる時とき、いつもいちばん後うしろの席せきに座すわります。
2. 看电影时，（我）总是坐在最后的座位上。
  • 映画えいが: [名词] 电影。
  • を: [格助詞]
    • 表示动作的对象：看（見る）的对象是“电影”。
  • 見みる: [动词]
    • 原型: 見る (miru)
    • 形式: 动词原形（辞書形）。
    • 含义: 看。
    • 属于上一段动词。
    • 見みる時とき: 动词原形 + 時，表示“做...的时候”。表示在看电影这个动作发生的时间段内。
  • 時とき: [名词] 时候，时间。
  • 、: [読点] 逗号，表示停顿。
  • いつも: [副詞] 总是，经常。
  • いちばん: [副詞] 最，第一。
  • 後うしろ: [名词] 后面。
  • の: [格助詞]
    • 连接名词，表示位置关系。後ろの席 意为“后面的座位”。
  • 席せき: [名词] 座位。
  • に: [格助詞]
    • 表示动作的着落点或归着点：表示“坐在”这个动作发生的具体位置“座位上”。
  • 座すわります: [动词]
    • 原型: 座る (suwaru)
    • 形式: 「座る」的ます形。
    • 含义: 坐。
    • 表示习惯性的动作（因为前面有「いつも」）。丁寧体。
    • 属于五段动词。
句意: 描述一个习惯：在看电影的时候，总是选择坐在最后排的座位上。
3. 李さんはテレビを見ながら食事をします。
3. 李りさんはテレビを見みながら食事しょくじをします。
3. 小李边看电视边吃饭。
  • 李りさん: [名词] 小李，李先生/女士。さん 是接尾词，表示尊敬。
  • は: [係助詞]
    • 提示主题: 表明句子谈论的对象是“李さん”。读作 わ (wa)。
  • テレビ: [名词] 电视。
  • を: [格助詞]
    • 表示动作的对象：看（見る）的对象是“电视”。
  • 見みながら: [动词 + 接続助詞]
    • 动词原型: 見る (miru)
    • 形式: 动词ます形去掉ます + ながら (見ます → 見 + ながら)。
    • 含义: 表示两个动作同时进行，“一边...一边...”。主要动作是后面的动作（食事をします），前面的动作（テレビを見る）是伴随性的。
    • 注意: 前后动作的主语必须是同一个人。
  • 食事しょくじ: [名词] 吃饭，用餐。
  • をします: [动词短语]
    • を: [格助詞] 表示动作的对象，这里指进行“食事”这个行为。
    • します:
      • 原型: する (suru)
      • 形式: 「する」的ます形。
      • 含义: 做，干。食事をする 是固定搭配，意为“吃饭”。
      • 表示习惯或当前的动作。丁寧体。
      • 属于サ行变格动词。
句意: 描述李先生/女士的习惯或行为：一边看电视一边吃饭。
4. 李さん、明日パーティーに行くでしょう？
4. 李りさん、明日あしたパーティーに行いくでしょう？
4. 小李，（你）明天参加联欢会吧？
  • 李りさん: [名词] 小李，李先生/女士。这里是称呼。
  • 、: [読点] 逗号，表示停顿或呼唤。
  • 明日あした: [名词] 明天。
  • パーティー: [名词] 聚会，联欢会 (Party)。
  • に: [格助詞]
    • 表示移动的目的地或方向：“去”的目的地是“パーティー”。
  • 行いく: [动词]
    • 原型: 行く (iku)
    • 形式: 动词原形（辞書形）。用于修饰后面的推量助动词「でしょう」。普通体形式。
    • 含义: 去。
    • 属于五段动词。
  • でしょう: [助动词]
    • 接在动词、形容词、名词等后面。这里接在动词普通体后面。
    • 表示推测或确认。句尾语调上扬时，表示向对方确认自己认为是事实的事情，期待对方同意。相当于汉语的“...吧？”。
    • 比「だろう」更礼貌。
  • ？: [标点符号] 问号。虽然语法上「でしょう」带有疑问/确认的含义，书写时也常加问号。
句意: 说话人向小李确认明天是否会去参加聚会，带有一定的推测和征求同意的语气。

応用おうよう

（到いたる分ぶん公司こうし后ご）
森：今朝、公園を散歩している時、大勢の人が集まっているのを見ました。
森：今朝けさ、公園こうえんを散歩さんぽしている時とき、大勢おおぜいの人ひとが集あつまっているのを見みました。
森：今晨，在公园散步时，看见有很多人聚集在那里。
  • 今朝けさ: [名词] 今天早上。
  • 、: [読点] 逗号。
  • 公園こうえん: [名词] 公园。
  • を: [格助詞]
    • 表示移动经过的场所：“散步”这个动作发生的范围是“公园”。
  • 散歩さんぽしている: [动词短语]
    • 原型: 散歩する (sanpo suru)
    • 形式: 「散歩する」的て形「散歩して」 + 「いる」。表示动作正在进行或状态的持续。这里指“正在散歩”的状态。
    • 属于サ行变格动词。
  • 時とき: [名词] 时候。散歩している時 指“正在散步的时候”。
  • 大勢おおぜい: [名词] 很多人（通常指人）。
  • の: [格助詞]
    • 连接名词：大勢の人 意为“很多的人”。
  • 人ひと: [名词] 人。
  • が: [格助詞]
    • 提示主语：表示“聚集”这个动作的主体是“大勢の人”。
  • 集あつまっている: [动词]
    • 原型: 集まる (atsumaru)
    • 形式: 「集まる」的て形「集まって」 + 「いる」。表示动作结果的状态持续，即“聚集着”的状态。
    • 属于五段动词。
  • の: [形式名词]
    • 将前面的动词短语「大勢の人が集まっている」名词化，使其成为「見ました」的宾语。相当于“...的事情”或“...的景象”。
  • を: [格助詞]
    • 表示动作的对象：看（見ました）的对象是「大勢の人が集まっているの」（很多人聚集着的景象）。
  • 見みました: [动词]
    • 原型: 見る (miru)
    • 形式: 「見る」的ます形「見ます」的过去式「見ました」。
    • 含义: 看见。表示过去的动作。丁寧体。
    • 属于上一段动词。
句意: 森先生说，今天早上在公园散步时，看到了很多人聚集在那里的景象。
李：ああ、お年寄りが多かったでしょう？
李：ああ、お年寄としよりが多おおかったでしょう？
李：哦，老年人居多吧？
  • ああ: [感叹词] 哦，啊。表示领会或轻微的惊讶。
  • 、: [読点] 逗号。
  • お年寄としより: [名词] 老年人。お 是接头词，表示尊敬或美化。
  • が: [格助詞]
    • 提示主语：表示“多”的主体是“お年寄り”。
  • 多おおかった: [形容词]
    • 原型: 多い (ooi)
    • 形式: イ形容词「多い」的过去式「多かった」。
    • 含义: 多的。
  • でしょう: [助动词]
    • 接在形容词过去式后面。
    • 表示推测或确认。句尾语调上扬，表示向对方确认推测，意为“...吧？”。
    • 比「だろう」更礼貌。
  • ？: [标点符号] 问号。
句意: 小李推测森先生看到的聚集人群中老年人应该很多，并向森先生确认。
森：ええ。太極拳やラジオ体操をしていました。
森：ええ。太極拳たいきょくけんやラジオ体操たいそうをしていました。
森：是的。（他们）在打太极拳或做广播体操。
  • ええ: [感叹词] 嗯，是的。表示肯定同意。比「はい」稍微口语化。
  • 。: [句点] 句号。
  • 太極拳たいきょくけん: [名词] 太极拳。
  • や: [並列助詞]
    • 连接名词，表示列举。暗示除了列举的事项（太极拳、广播体操）之外，可能还有其他类似活动。相当于汉语的“...啦...之类的”。
  • ラジオ体操たいそう: [名词] 广播体操。
  • を: [格助詞]
    • 表示动作的对象：进行（していました）的对象是“太極拳やラジオ体操”。
  • していました: [动词]
    • 原型: する (suru)
    • 形式: 「する」的て形「して」 + 「いました」（「いる」的ます形过去式）。表示过去某个时间点动作正在进行或持续的状态。这里指当时（今朝）他们正在做这些活动。
    • 太極拳をする、ラジオ体操をする 是固定搭配。
    • 丁寧体。
    • 属于サ行变格动词。
句意: 森先生肯定了李先生的猜测，并说明了那些老年人当时正在进行的活动，如打太极拳、做广播体操等。
李：朝の運動ですよ。社交ダンスをしている人たちもいたでしょう？
李：朝あさの運動うんどうですよ。社交しゃこうダンスをしている人ひとたちもいたでしょう？
李：是晨练啊。也有跳交际舞的人们吧？
  • 朝あさ: [名词] 早上。
  • の: [格助詞]
    • 连接名词，表示所属或性质。朝の運動 意为“早上的运动”，即晨练。
  • 運動うんどう: [名词] 运动。
  • ですよ: [判断助动词 + 終助詞]
    • です: [助动词] 丁寧体的判断助动词，表示肯定判断。
    • よ: [終助詞] 加在句末，起强调、告知对方其不知道的信息或提醒的作用。
    • ですよ 整体表示“是...啊/呀”，带有告知和轻微强调的语气。
  • 。: [句点] 句号。
  • 社交しゃこうダンス: [名词] 交际舞 (Social Dance)。
  • を: [格助詞]
    • 表示动作的对象：做（している）的对象是“社交ダンス”。
  • している: [动词]
    • 原型: する (suru)
    • 形式: 「する」的て形「して」 + 「いる」。表示动作正在进行或状态持续。社交ダンスをしている 指“正在跳交际舞”。
  • 人ひとたち: [名词] 人们。たち 是接尾词，表示复数。
  • も: [係助詞]
    • 表示类推、补充，“也”。暗示除了练太极拳和广播体操的人，还有跳交际舞的人。
  • いた: [动词]
    • 原型: いる (iru)
    • 形式: 「いる」的过去式「いた」。普通体。用于修饰后面的推量助动词「でしょう」。
    • 含义: 表示（有生命物体，主要是人或动物）的存在。这里指“有（人）”。
    • 属于上一段动词。
  • でしょう: [助动词]
    • 接在动词普通体过去式后面。
    • 表示推测或确认。句尾语调上扬，表示向对方确认推测，“...吧？”。
  • ？: [标点符号] 问号。
句意: 小李解释说那是晨练，并进一步推测，向森先生确认是否也有人在跳交际舞。
森：ええ、いました。ほかに、踊りながら歌を歌っている人もいましたよ。
森：ええ、いました。ほかに、踊おどりながら歌うたを歌うたっている人ひともいましたよ。
森：是的，有。此外，还有边跳舞边唱歌的人。
  • ええ: [感叹词] 嗯，是的。表示肯定。
  • 、: [読点] 逗号。
  • いました: [动词]
    • 原型: いる (iru)
    • 形式: 「いる」的ます形「います」的过去式「いました」。
    • 含义: 表示（人或动物）的存在。这里回答李先生的问题，表示“有（跳交际舞的人）”。丁寧体。
  • 。: [句点] 句号。
  • ほかに: [副詞] 另外，此外。
  • 、: [読点] 逗号。
  • 踊おどりながら: [动词 + 接続助詞]
    • 动词原型: 踊る (odoru)
    • 形式: 动词ます形去掉ます + ながら (踊ります → 踊り + ながら)。
    • 含义: 一边跳舞一边...。表示两个动作同时进行。
    • 属于五段动词。
  • 歌うた: [名词] 歌。
  • を: [格助詞]
    • 表示动作的对象：唱（歌っている）的对象是“歌”。
  • 歌うたっている: [动词]
    • 原型: 歌う (utau)
    • 形式: 「歌う」的て形「歌って」 + 「いる」。表示动作正在进行或状态持续。这里指“正在唱歌”的状态。
    • 属于五段动词。
  • 人ひと: [名词] 人。踊りながら歌を歌っている人 指“一边跳舞一边唱歌的人”。
  • も: [係助詞]
    • 表示类推、补充，“也”。表示除了之前提到的人之外，还有这样的人。
  • いました: [动词]
    • 原型: いる (iru)
    • 形式: 「いる」的ます形「います」的过去式「いました」。
    • 含义: 表示存在，“也有（这样的人）”。丁寧体。
  • よ: [終助詞]
    • 加在句末，起强调、告知对方新信息或提醒的作用。
  • 。: [句点] 句号。
句意: 森先生肯定了有跳交际舞的人，并补充说此外还有一边跳舞一边唱歌的人。
（小戴しょうたい想おもい起おこす孩提がいてい時代じだいの事こと情じょう）
戴：そう言えば、小さい時、よく祖母といっしょに公園へ行きました。
戴：そう言いえば、小ちいさい時とき、よく祖母そぼといっしょに公園こうえんへ行いきました。
戴：说起来，（我）小时候常和祖母一起去公园。
  • そう言いえば: [惯用语] 说起来，这么说来。用于根据对方的话或当前话题联想起某事。
  • 、: [読点] 逗号。
  • 小ちいさい: [形容词]
    • 原型: 小さい (chiisai)
    • 形式: イ形容词原形。
    • 含义: 小的。小さい時 意为“小的时候”，即“小时候”。
  • 時とき: [名词] 时候。
  • 、: [読点] 逗号。
  • よく: [副詞] 经常，常常。
  • 祖母そぼ: [名词] 祖母，奶奶，外婆。
  • と: [格助詞]
    • 表示共同动作的对象，“和...”。
  • いっしょに: [副詞] 一起。
  • 公園こうえん: [名词] 公园。
  • へ: [格助詞]
    • 表示移动的方向或目的地。读作 え (e)。与 に 在表示目的地时很多情况下可以互换，但 へ 更侧重方向。
  • 行いきました: [动词]
    • 原型: 行く (iku)
    • 形式: 「行く」的ます形「行きます」的过去式「行きました」。
    • 含义: 去。表示过去的习惯性动作（因为有「よく」）。丁寧体。
句意: 小戴由刚才的话题联想到，说起这个，他小时候经常和祖母一起去公园。
森：へえ、いっしょに運動をしたんですか。
森：へえ、いっしょに運動うんどうをしたんですか。
森：哦，（你和祖母）一起锻炼吗？
  • へえ: [感叹词] 哦，嗬。表示惊讶或感兴趣。
  • 、: [読点] 逗号。
  • いっしょに: [副詞] 一起。
  • 運動うんどう: [名词] 运动。
  • を: [格助詞]
    • 表示动作的对象：进行（した）的对象是“運動”。運動をする 是“做运动”。
  • した: [动词]
    • 原型: する (suru)
    • 形式: 「する」的过去式「した」。普通体。
    • 含义: 做了。
  • ん: [形式名词/助词]
    • 「の」的口语音变。接在用言连体形（这里是动词过去式「した」）后。
    • んです (或普通体 んだ) 用于解释原因、理由、状况，或表示强调、关心、追问等。这里带有探寻确认的语气。
  • です: [助动词] 丁寧体的判断助动词。
  • か: [終助詞] 表示疑问。
句意: 森先生听了小戴的话，好奇地询问小戴是否和祖母一起做运动。
戴：いいえ。わたしは遊びながら祖母が太極拳をするのを見ていました。

  • いいえ: [感叹词] 不，不是。表示否定。
  • 。: [句点] 句号。
  • わたし: [代名词] 我。
  • は: [係助詞]
    • 提示主题。读作 わ (wa)。
  • 遊あそびながら: [动词 + 接続助詞]
    • 动词原型: 遊ぶ (asobu)
    • 形式: 动词ます形去掉ます + ながら (遊びます → 遊び + ながら)。
    • 含义: 一边玩一边...。表示两个动作同时进行，主要动作是后面的「見ていました」。

  • 祖母そぼ: [名词] 祖母。
  • が: [格助詞]
    • 提示从句中的主语：表示“打太极拳”这个动作的主体是“祖母”。
  • 太極拳たいきょくけん: [名词] 太极拳。
  • を: [格助詞]
    • 表示动作的对象：做（する）的对象是“太極拳”。
  • する: [动词]
    • 原型: する (suru)
    • 形式: 动词原形（辞書形）。用于修饰后面的形式名词「の」。

    • 丁寧体。
句意: 小戴否定了和祖母一起锻炼，说明自己当时是一边玩耍，一边看着祖母打太极拳。
