第6課 吉田さんは 来月 中国へ 行きます

基本きほん

1. 吉田さんは 来月 中国へ 行きます。
1. 吉田よしださんは 来月らいげつ 中国ちゅうごくへ 行いきます。
1. 吉田先生下个月去中国。
  • 吉田よしだ: [固有名詞] 日本姓氏，吉田。
  • さん: [接尾語]
    • 接在人名或表示人的名词后，表示尊敬或礼貌。男女通用。
    • 相当于汉语的“先生”、“女士”或称呼本身。
  • は: [係助詞]
    • 提示主题: 表明句子谈论的对象是“吉田さん”。“关于吉田先生...”。
    • 读作 わ (wa)。
  • 来月らいげつ: [名詞] 下个月。表示时间。
  • 中国ちゅうごく: [固有名詞] 中国。
  • へ: [格助詞]
    • 表示移动行为的目的地或方向。
    • 通常接在表示场所的名词后，与表示移动的动词（如 行きます、来ます、帰ります）连用。
    • 读作 え (e)。
  • 行いきます: [動詞]
    • 原型: 行く (いく) [五段活用动词]
    • 形式: ます形 (ますけい - Polite Form)。用于构成礼貌的陈述句或疑问句。
    • 含义: 去。表示向某个方向移动。
句意: 陈述吉田先生下个月要去中国这一事实或计划，语气礼貌。
2. 李さんは 先月 北京から 来ました。
2. 李りさんは 先月せんげつ 北京ペキンから 来きました。
2. 小李上个月从北京来。
  • 李り: [固有名詞] 姓氏，李。
  • さん: [接尾語] 接在人名后表示尊敬或礼貌。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 先月せんげつ: [名詞] 上个月。表示过去的时间。
  • 北京ペキン: [固有名詞] 北京。
  • から: [格助詞]
    • 表示动作的起点（时间或空间）。这里表示空间起点，“从北京”。
  • 来きました: [動詞]
    • 原型: 来る (くる) [カ行变格活用动词]
    • 形式: ます形的过去式 (ました)。表示过去的动作或状态，语气礼貌。
    • 含义: 来。表示从别处向说话人所在地移动。
句意: 陈述小李上个月从北京来到这里的这个过去的动作，语气礼貌。
3. 小野さんは 友達と 帰りました。
3. 小野おのさんは 友達ともだちと 帰かえりました。
3. 小野女士和朋友(一块儿)回去了。
  • 小野おの: [固有名詞] 日本姓氏，小野。
  • さん: [接尾語] 接在人名后表示尊敬或礼貌。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 友達ともだち: [名詞] 朋友。
  • と: [格助詞]
    • 表示共同进行动作的对象，“和...一起”。
  • 帰かえりました: [動詞]
    • 原型: 帰る (かえる) [五段活用动词]
    • 形式: ます形的过去式 (ました)。表示过去的动作或状态，语气礼貌。
    • 含义: 回去，回家。指回到原来的地方（家、公司、国家等）。
句意: 陈述小野女士和朋友一起回去了这个过去的动作，语气礼貌。
4. 森さんは 東京から 広島まで 新幹線で 行きます。
4. 森もりさんは 東京とうきょうから 広島ひろしままで 新幹線しんかんせんで 行いきます。
4. 森先生从东京坐新干线去广岛。
  • 森もり: [固有名詞] 日本姓氏，森。
  • さん: [接尾語] 接在人名后表示尊敬或礼貌。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 東京とうきょう: [固有名詞] 东京。
  • から: [格助詞] 表示动作的起点（空间），“从东京”。
  • 広島ひろしま: [固有名詞] 广岛。
  • まで: [副助詞] 表示动作的终点（空间），“到广岛”。
  • 新幹線しんかんせん: [名詞] 新干线（日本高速铁路）。
  • で: [格助詞]
    • 表示交通手段或工具，“乘坐...”、“用...”。这里指乘坐新干线。
  • 行いきます: [動詞]
    • 原型: 行く (いく) [五段活用动词]
    • 形式: ます形。表示现在或将来的动作，语气礼貌。
    • 含义: 去。
句意: 陈述森先生将乘坐新干线从东京去往广岛的计划或事实，语气礼貌。

(早晨，在办公室)
吉田: 李さん、昨日は 何時に アパートへ 帰りましたか。
吉田: 李りさん、昨日きのうは 何時なんじに アパートへ 帰かえりましたか。
小李，昨天(你)几点回公寓的?
  • 李りさん: [固有名詞 + 接尾語] 对李先生/女士的称呼。
  • 昨日きのう: [名詞] 昨天。
  • は: [係助詞] 提示主题“昨天”。读作 わ (wa)。
  • 何時なんじ: [疑問詞] 几点。询问具体时间。
  • に: [格助詞] 表示动作发生的具体时间点，“在几点”。
  • アパート: [名詞] 公寓 (Apartment)。
  • へ: [格助詞] 表示移动的目的地或方向。读作 え (e)。
  • 帰かえりました: [動詞]
    • 原型: 帰る (かえる)
    • 形式: ます形的过去式 (ました)。
    • 含义: 回去，回家。
  • か: [終助詞] 表示疑问。
句意: 吉田询问李昨天回公寓的具体时间。
李: ええと、たしか 11時半ごろです。
李: ええと、たしか 11時じ半はんごろです。
哦，好像是11点半左右。
  • ええと: [感動詞] 嗯，哦。用于思考或回忆时的发语词。
  • たしか: [副詞] 的确，好像是。表示不完全确定的记忆或推测。
  • 11時じ半はん: [名詞] 十一点半。
  • ごろ: [接尾語]
    • 接在表示时间的名词后，表示大致的时间范围，“...左右”。
  • です: [助動詞] 丁寧体的判断助动词，表示肯定判断。
句意: 李回忆并回答大概是昨天晚上11点半左右回的公寓。

  • か: [終助詞] 表示疑问。
  • タクシー: [名詞] 出租车 (Taxi)。
  • です: [助動詞] 判断助动词。
  • か: [終助詞] 表示疑问。这里构成 "名词 + ですか" 的形式，询问“是...吗？”。

  • 行いきました: [動詞]

  • 駅えき: [名詞] 车站。

  • アパート: [名詞] 公寓。
  • まで: [副助詞] 表示终点，“到公寓”。

小野: 何時なんじに うちへ 帰かえりましたか。
(那你们)几点回的家?
  • 何時なんじ: [疑問詞] 几点。
  • に: [格助詞] 表示动作发生的具体时间点。
  • うち: [名詞] 家。
  • へ: [格助詞] 表示移动的目的地。读作 え (e)。
  • 帰かえりました: [動詞]
    • 原型: 帰る (かえる)
    • 形式: ます形的过去式 (ました)。
    • 含义: 回去，回家。
  • か: [終助詞] 表示疑问。
句意: 小野询问森和科长昨晚回家的具体时间。
森: 真夜中の 2時です。
森: 真夜中まよなかの 2時じです。
深夜两点。
  • 真夜中まよなか: [名詞] 深夜，半夜。
  • の: [格助詞] 连接名词，表示所属或修饰关系。“深夜的两点”。
  • 2時じ: [名詞] 两点。
  • です: [助動詞] 表示肯定判断。
句意: 森回答他们是深夜两点才回家的。
李: 2時ですか。それは お疲れ様でした。
李: 2時じですか。それは お疲つかれ様さまでした。
两点啊。够累的吧。
  • 2時じ: [名詞] 两点。
  • です: [助動詞] 判断助动词。
  • か: [終助詞] 表示疑问或感叹，这里带有惊讶和确认的语气。
  • それは: [指示代名詞 + 係助詞] “那个（指深夜两点回家这件事）”。それ指代前面提到的情况。は提示主题。
  • お疲つかれ様さまでした: [慣用句] 辛苦了。
    • お: [接頭語] 美化语，表示尊敬或郑重。
    • 疲れ (つかれ): [名詞] 疲劳。动词疲れる(つかれる)的连用形名词化。
    • 様 (さま): [接尾語] 表示尊敬。
    • でした: [助動詞] です的过去式。用于对过去的事情表示慰问。
    • 常用于工作结束、或慰劳对方的辛劳时。
句意: 李听到是两点回家表示惊讶，并对森表示慰问，觉得他一定很累。

基本

1. 吉田先生下个月去中国。

2. 小李上个月从北京来。

3. 小野女士和朋友(一块儿)回去了。

4. 森先生从东京坐新干线去广岛。

基本对话

A 甲: (你)什么时候去美国?

乙: 10月份去。

B 甲: (你)怎么去车站?

乙: 骑车去。

C 甲: (你)和谁去美术馆?

乙: 和朋友去。

D 甲: (你)是乘飞机从大阪去上海吗?

乙: 不，坐渡轮去。

応用

(早晨，在办公室)

吉田: 小李，昨天(你)几点回公寓的?

李: 哦，好像是11点半左右。

吉田: 怎么回去的，坐出租车?

李: 坐电车回去的。坐电车到涩谷，然后从车站走回公寓去。

吉田: 小野，你呢?

小野: 我也是坐电车。从车站是坐出租车回家的。

李: 几点回去的?

小野: 12点左右。

(小野向正在打瞌睡的森搭话)

小野: 森，(你)昨晚直接回家了吗?

森: 没有，我和科长一起去银座了。

李: 啊? 银座?

小野: (那你们)几点回的家?

森: 深夜两点。

李: 两点啊。够累的吧。
