第18課 携帯電話は とても 小さくなりました

基本きほん

1. 携帯電話は とても 小さくなりました。
1. 携帯けいたい電話でんわは とても 小ちいさくなりました。
1. 手机变小了。
  • 携帯けいたい電話でんわ: [名词] 手机，移动电话。
  • は: [係助詞 (かかりじょし)]
    • 提示主题: 表明句子谈论的对象是“携帯電話”。
    • 读作 わ (wa)。
  • とても: [副词] 非常，很。用于修饰形容词或状态。
  • 小ちいさく: [形容词连用形]
    • 原型: 小ちいさい (小的) [い形容词]
    • 形式: 将词尾「い」变为「く」。
    • 含义: 用于连接动词（特别是「なる」、「する」）或作副词修饰动词。
    • 在此句中，与「なる」连接，表示状态的变化。
  • なりました: [动词 + 助动词]
    • なり:
      • 原型: なる (变成) [五段动词]
      • 形式: 动词连用形 (ます形去掉ます)。
    • ました: [助动词]
      • 表示过去的动作或状态，且语气礼貌（丁寧体）。
    • 结构: 形容词连用形(く) + なる → 表示状态的自然变化。“变得...”
    • 含义: 变得...了 (过去式)。
句意: 手机变得非常小了。（描述过去发生的变化）
2. テレビの音を大きくします。
2. テレビの音おとを 大おおきくします。
2. 把电视机的声音开大一点儿。
  • テレビ: [名词] 电视机。
  • の: [格助詞]
    • 表示所属或修饰关系。“电视机的...”
  • 音おと: [名词] 声音。
  • を: [格助詞]
    • 表示动作（します）的直接对象。“把声音...”
  • 大おおきく: [形容词连用形]
    • 原型: 大おおきい (大的) [い形容词]
    • 形式: 将词尾「い」变为「く」。
    • 含义: 用于连接动词。在此句中连接「する」。
  • します: [动词 + 助动词]
    • し:
      • 原型: する (做；使之成为...) [サ行变格动词]
      • 形式: 动词连用形 (ます形去掉ます)。
    • ます: [助动词]
      • 表示现在或未来的动作或状态，且语气礼貌（丁寧体）。
    • 结构: 形容词连用形(く) + する → 表示主观意志，人为地使某物达到某种状态。“把...弄成...”
    • 含义: 使之变大，调大。
句意: （我）要把电视机的声音调大。（表示主观意愿或即将进行的操作）
3. 息子は医者になりました。
3. 息子むすこは 医者いしゃになりました。
3. 儿子当医生了。
  • 息子むすこ: [名词] 儿子。
  • は: [係助詞]
    • 提示主题: 表明句子谈论的对象是“息子”。
    • 读作 わ (wa)。
  • 医者いしゃ: [名词] 医生。
  • に: [格助詞]
    • 表示变化的结果。当与动词「なる」(变成) 连用，前面接名词或形容动词词干时，表示变成了该名词或形容动词所表示的状态或身份。
  • なりました: [动词 + 助动词]
    • なり:
      • 原型: なる (变成) [五段动词]
      • 形式: 动词连用形 (ます形去掉ます)。
    • ました: [助动词]
      • 表示过去的动作或状态，且语气礼貌。
    • 结构: 名词 + に + なる → 表示身份、状态的转变。“成为...”。
    • 含义: 变成了医生 (过去式)。
句意: （我的）儿子成为了一名医生。（描述过去发生的变化）
4. 部屋を きれいに してください。
4. 部屋へやを きれいに してください。
4. 请把房间打扫干净。
  • 部屋へや: [名词] 房间。
  • を: [格助詞]
    • 表示动作（します）的直接对象。“把房间...”
  • きれいに: [形容动词连用形]
    • 原型: きれいだ (干净的，漂亮的) [な形容词/形容动词]
    • 形式: 将词尾「だ」变为「に」。
    • 含义: 用于连接动词（特别是「する」）或作副词修饰动词。在此句中连接「する」。
  • して: [动词て形]
    • 原型: する (做；使之成为...) [サ行变格动词]
    • 形式: 动词て形。用于连接后续的补助动词或表示动作的接续。
    • 结构: 形容动词连用形(に) + する → 表示主观意志，人为地使某物达到某种状态。“把...弄成...”。
  • ください: [补助动词]
    • 接在动词て形后面，表示请求对方做某事。“请...”
    • 结构: 动词て形 + ください → “请做某事”。
句意: 请把房间弄干净/打扫干净。（表示请求）

A 甲: 風邪は どうですか。
A 甲: 風邪かぜは どうですか。
A 甲: 感冒怎么样了？
  • 風邪かぜ: [名词] 感冒。
  • は: [係助詞]
    • 提示主题: 表明谈论的是“感冒”的情况。
    • 读作 わ (wa)。
  • どう: [副词/疑问词] 怎么样，如何。用来询问状态或情况。
  • です: [助动词]
    • 丁寧体判断助动词，使句子礼貌。
  • か: [終助詞]
    • 加在句末，表示疑问。
句意: （你的）感冒情况怎么样？
乙: ゆうべ 薬を 飲みましたが、まだ よくなりません。
乙: ゆうべ 薬くすりを 飲のみましたが、まだ よくなりません。
乙: 昨晚吃了点儿药，可还不见好。
  • ゆうべ: [名词/副词] 昨晚，昨天晚上。
  • 薬くすり: [名词] 药。
  • を: [格助詞]
    • 表示动作（飲みました）的直接对象。
  • 飲のみましたが: [动词ます形 + 助词]
    • 飲のみ:
      • 原型: 飲のむ (喝，服用) [五段动词]
      • 形式: 动词连用形 (ます形去掉ます)。
    • ました: [助动词] 表示过去的动作，语气礼貌。
    • が: [接続助詞]
      • 表示转折关系，相当于汉语的“但是”、“可是”。连接前后两个句子。
  • まだ: [副词] 还，仍然。表示状况、程度没有变化或尚未达到。
  • よく: [形容词连用形]
    • 原型: よい / いい (好的) [い形容词 - 特殊变化]
    • 形式: 连用形为「よく」。
    • 含义: 好地。在此处与「なる」连用。
  • なりません: [动词 + 助动词]
    • なり:
      • 原型: なる (变成) [五段动词]
      • 形式: 动词连用形。
    • ません: [助动词]
      • 表示现在或未来的否定，语气礼貌。
    • 结构: 形容词连用形(く) + なる → 变得...
    • 含义: よくなりません 意思是“没有变好”，“不见好转”。
句意: 昨晚吃了药，但是（病情）还没有好转。
B 甲: 旅行の荷物は 軽いほうが いいですよ。
B 甲: 旅行りょこうの 荷物にもつは 軽かるいほうが いいですよ。
B 甲: 去旅行(的时候)行李还是轻点儿好。
  • 旅行りょこう: [名词] 旅行。
  • の: [格助詞] 表示所属或相关。“旅行的...”
  • 荷物にもつ: [名词] 行李。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 軽かるい: [形容词] 轻的（重量）。
  • ほう: [名词] 方，方面；（两者比较时）...一方。
  • が: [格助詞]
    • 在这里强调前面的「軽いほう」是主语，常与「いい」连用构成「～ほうがいい」句型。
  • いい: [形容词] 好的。
  • です: [助动词] 丁寧体判断助动词。虽然口语中「いいです」有时省略「です」，但书面或更完整的表达会加上。此处省略了「です」。
  • よ: [終助詞]
    • 表示告知对方其不知道的信息，或加强语气。
  • 结构: 形容词简体 + ほうが いい (です) → 表示比较后的建议或判断。“...的一方更好”，“还是...点好”。
句意: （关于）旅行的行李，轻一点的比较好哦。（提出建议）
乙: はい。できるだけ 軽く します。

  • 軽かるく: [形容词连用形]

    • 形式: 词尾「い」变为「く」。
    • 含义: 连接动词「する」。
  • します: [动词 + 助动词]

      • 形式: 动词连用形。
    • ます: [助动词] 表示现在或将来的动作，语气礼貌。
    • 结构: 形容词连用形(く) + する → 人为地使之变成某种状态。“弄成...”。
    • 含义: 軽くします 意思是“（我）会把它弄轻”。
句意: 好的。我会尽量把（行李）弄轻一点。
