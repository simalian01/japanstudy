第5課 森さんは７時に起きます

基本きほん

1. 今 4時です。
1. 今いま ４よ時じです。
1. 现在4点。
  • 今いま: [名词/副词] 现在。
  • ４よ時じ: [名词] 4点钟。
    • 表示时间的表达方式。注意「４時」读作「よじ」。
  • です: [助动词]
    • 丁寧体 (ていねいたい - Polite Style) 的判断助动词。用于名词后，表示肯定判断或陈述事实。
    • 相当于汉语的“是...”。
    • 使整个句子变得礼貌。
句意: 陈述当前的时间是4点。
2. 森さんは 7時に 起きました。
2. 森もりさんは ７しち時じに 起おきました。
2. 森先生7点起床。
  • 森もりさん: [名词] 森先生/女士。
    • さん 是接在人名后的敬称，男女通用。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“森さん”。
    • 读作 わ (wa)。
  • ７しち時じ: [名词] 7点钟。
    • 表示时间的表达方式。注意「７時」读作「しちじ」。
  • に: [格助詞 (かくじょし)]
    • 表示动作发生的具体时间点。附在表示时间的名词后。
    • 含义: 在... (时间点)。
  • 起おきました: [动词]
    • 原型: 起おきる (一段动词 - Ichidan Verb)
    • 形式: ます形 (Masu form) 的过去式。
    • 含义: 起床 (的动作已完成)。
    • 结构: 动词ます形词干 (起き) + ました (表示过去时态的丁寧体助动词)。
句意: 森先生在7点钟起床了。(描述过去发生的动作)
3. 森さんは 先週 休みました。
3. 森もりさんは 先週せんしゅう 休やすみました。
3. 森先生上周休息。
  • 森もりさん: [名词] 森先生/女士。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“森さん”。
    • 读作 わ (wa)。
  • 先週せんしゅう: [名词] 上周。
    • 表示相对时间的词，通常后面不加助词「に」表示动作发生的时间。
  • 休やすみました: [动词]
    • 原型: 休やすむ (五段动词 - Godan Verb)
    • 形式: ます形 (Masu form) 的过去式。
    • 含义: 休息了 / 请假了。
    • 结构: 动词ます形词干 (休み) + ました (表示过去时态的丁寧体助动词)。
句意: 森先生上个星期休息了。
4. わたしは 昨日 働きませんでした。
4. わたしは 昨日きのう 働はたらきませんでした。
4. 我昨天没上班。
  • わたし: [代名词] 我 (性别中性，较常用)。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“わたし”。
    • 读作 わ (wa)。
  • 昨日きのう: [名词] 昨天。
    • 表示相对时间的词，通常后面不加助词「に」表示动作发生的时间。
  • 働はたらきませんでした: [动词]
    • 原型: 働はたらく (五段动词 - Godan Verb)
    • 形式: ます形 (Masu form) 的过去否定式。
    • 含义: 没有工作 / 没有上班。
    • 结构: 动词ます形词干 (働き) + ませんでした (表示过去否定时态的丁寧体助动词)。
句意: 我昨天没有工作（没上班）。

基本きほん会話かいわ

A 甲: 毎日、何時に 寝ますか。
A 甲: 毎日まいにち、何時なんじに 寝ねますか。
A 甲: （你）每天几点睡？
  • 毎日まいにち: [名词/副词] 每天。表示动作发生的频率或习惯性时间。
  • 何時なんじ: [疑问代词] 几点。用来询问具体时间。
  • に: [格助詞 (かくじょし)]
    • 表示动作发生的具体时间点。
  • 寝ねます: [动词]
    • 原型: 寝ねる (一段动词 - Ichidan Verb)
    • 形式: ます形 (Masu form)。
    • 含义: 睡觉。
    • 结构: 动词ます形词干 (寝) + ます (表示现在或未来时态，或习惯性动作的丁寧体助动词)。
  • か: [終助詞 (しゅうじょし)]
    • 加在句末，表示疑问。
句意: 询问对方每天在几点钟睡觉。
乙: 11時30分に 寝ます。
乙: １１じゅういち時じ３０さんじゅっ分ぷんに 寝ねます。
乙: 11点30分睡。
  • １１じゅういち時じ３０さんじゅっ分ぷん: [名词] 11点30分。
    • 表示具体时间。「分」读「ぷん」或「ふん」，此处「30分」通常读「さんじゅっぷん」。也可以说「11時半 (じゅういちじはん)」。
  • に: [格助詞 (かくじょし)]
    • 表示动作发生的具体时间点。
  • 寝ねます: [动词]
    • 原型: 寝ねる
    • 形式: ます形。
    • 含义: 睡觉。
    • 结构: 寝 + ます。
句意: 回答说在11点30分睡觉。
B 甲: 昨日、何時から 何時まで 働きましたか。
B 甲: 昨日きのう、何時なんじから 何時なんじまで 働はたらきましたか。
B 甲: 昨天（你）从几点工作到几点？
  • 昨日きのう: [名词] 昨天。
  • 何時なんじ: [疑问代词] 几点。
  • から: [格助詞 (かくじょし)]
    • 表示时间或空间的起点。含义: 从...。
  • 何時なんじ: [疑问代词] 几点。
  • まで: [副助詞 (ふくじょし)]
    • 表示时间或空间的终点。含义: 到...为止。
  • 働はたらきました: [动词]
    • 原型: 働はたらく
    • 形式: ます形的过去式。
    • 含义: 工作了。
    • 结构: 働き + ました。
  • か: [終助詞 (しゅうじょし)]
    • 表示疑问。
句意: 询问对方昨天工作的时间段（从开始到结束）。
乙: 9時から 6時まで 働きました。
乙: ９く時じから ６ろく時じまで 働はたらきました。
乙: 从9点工作到6点。
  • ９く時じ: [名词] 9点钟。
    • 注意「９時」读作「くじ」。
  • から: [格助詞 (かくじょし)]
    • 表示时间起点。
  • ６ろく時じ: [名词] 6点钟。
  • まで: [副助詞 (ふくじょし)]
    • 表示时间终点。
  • 働はたらきました: [动词]
    • 原型: 働はたらく
    • 形式: ます形的过去式。
    • 含义: 工作了。
    • 结构: 働き + ました。
句意: 回答说从早上9点工作到下午6点。
C 甲: 先週 休みましたか。
C 甲: 先週せんしゅう 休やすみましたか。
C 甲: （你）上周休息了吗？
  • 先週せんしゅう: [名词] 上周。
  • 休やすみました: [动词]
    • 原型: 休やすむ
    • 形式: ます形的过去式。
    • 含义: 休息了 / 请假了。
    • 结构: 休み + ました。
  • か: [終助詞 (しゅうじょし)]
    • 表示疑问。
句意: 询问对方上个星期是否休息了。
乙: いいえ、休みませんでした。
乙: いいえ、休やすみませんでした。
乙: 没休息。
  • いいえ: [感叹词] 不，不是。用于否定回答。
  • 休やすみませんでした: [动词]
    • 原型: 休やすむ
    • 形式: ます形的过去否定式。
    • 含义: 没有休息 / 没有请假。
    • 结构: 休み + ませんでした。
句意: 否定回答，表示上周没有休息。
D 甲: 試験は いつ 始まりますか。
D 甲: 試験しけんは いつ 始はじまりますか。
D 甲: 什么时候开始考试？
  • 試験しけん: [名词] 考试。
  • は: [係助詞 (かかりじょし)]
    • 提示主题。
  • いつ: [疑问代词] 什么时候。用来询问时间（不一定是具体几点，可以是日期、时段等）。
  • 始はじまります: [动词]
    • 原型: 始はじまる (五段动词 - Godan Verb, 不及物动词 Intransitive)
    • 形式: ます形。
    • 含义: 开始。
    • 结构: 始まり + ます。
  • か: [終助詞 (しゅうじょし)]
    • 表示疑问。
句意: 询问考试开始的时间。
乙: 来週の 木曜日です。
乙: 来週らいしゅうの 木曜日もくようびです。
乙: 下周四。
  • 来週らいしゅう: [名词] 下周。
  • の: [格助詞 (かくじょし)]
    • 连接名词，表示所属、性质等关系。这里连接两个时间名词，表示“下周的星期四”。
  • 木曜日もくようび: [名词] 星期四。
  • です: [助动词]
    • 丁寧体的判断助动词，表示肯定。
    • 省略了主语和动词（試験は 来週の 木曜日に 始まります）。
句意: 回答考试的时间是下个星期四。

応用おうよう

(10点已过，吉田问小野)
吉田: 小野さん、李さんの 歓迎会は いつですか。
吉田: 小野おのさん、李りさんの 歓迎会かんげいかいは いつですか。
吉田: 小野，李季的欢迎会是什么时候？
  • 小野おのさん: [名词] 小野先生/女士。
  • 李りさん: [名词] 李先生/女士。
  • の: [格助詞 (かくじょし)]
    • 表示所属关系，“李先生/女士 的”。
  • 歓迎会かんげいかい: [名词] 欢迎会。
  • は: [係助詞 (かかりじょし)]
    • 提示主题，“关于欢迎会”。
  • いつ: [疑问代词] 什么时候。
  • です: [助动词]
    • 丁寧体的判断助动词。
  • か: [終助詞 (しゅうじょし)]
    • 表示疑问。
句意: 吉田向小野询问李先生/女士的欢迎会在什么时候举行。
小野: あさっての 夜です。
小野: あさっての 夜よるです。
小野: 后天晚上。
  • あさって: [名词] 后天。
  • の: [格助詞 (かくじょし)]
    • 连接名词，表示“后天的”。
  • 夜よる: [名词] 晚上。
  • です: [助动词]
    • 丁寧体的判断助动词。
句意: 小野回答说欢迎会在后天晚上。
吉田: 何時からですか。
吉田: 何時なんじからですか。
吉田: 几点开始？
  • 何時なんじ: [疑问代词] 几点。
  • から: [格助詞 (かくじょし)]
    • 表示起点，“从...”。暗示“从几点开始”。
  • です: [助动词]
    • 丁寧体的判断助动词。
  • か: [終助詞 (しゅうじょし)]
    • 表示疑问。
句意: 吉田进一步询问欢迎会开始的具体时间。
小野: 6時からです。
小野: ６ろく時じからです。
小野: 6点开始。
  • ６ろく時じ: [名词] 6点钟。
  • から: [格助詞 (かくじょし)]
    • 表示起点，“从...”。
  • です: [助动词]
    • 丁寧体的判断助动词。
句意: 小野回答说欢迎会从6点开始。
(森悄悄走进办公室)
吉田: 森君、おはよう。今 何時ですか。
吉田: 森もり君くん、おはよう。今いま 何時なんじですか。
吉田: 森君，你早。现在几点了？
  • 森もり君くん: [名词] 森君。
    • 君 (くん) 是接在男性（特别是下属或同辈）名字后的称呼，比「さん」随便一些。
  • おはよう: [寒暄语] 早上好 (较随便的说法，完整形式是 おはようございます)。
  • 今いま: [名词/副词] 现在。
  • 何時なんじ: [疑问代词] 几点。
  • です: [助动词]
    • 丁寧体的判断助动词。
  • か: [終助詞 (しゅうじょし)]
    • 表示疑问。
句意: 吉田向森君打招呼并询问当前时间。
森: 10時15分です。
森: １０じゅう時じ１５じゅうご分ふんです。
森: 10点15分。
  • １０じゅう時じ１５じゅうご分ふん: [名词] 10点15分。
    • 「15分」读作「じゅうごふん」。
  • です: [助动词]
    • 丁寧体的判断助动词。
句意: 森君回答当前时间是10点15分。
吉田: 遅刻ですね。
吉田: 遅刻ちこくですね。
吉田: 迟到了吧。
  • 遅刻ちこく: [名词/する动词] 迟到。
  • です: [助动词]
    • 丁寧体的判断助动词。
  • ね: [終助詞 (しゅうじょし)]
    • 加在句末，表示确认、征求同意或感叹。这里带有确认“你迟到了”的语气。
句意: 吉田指出森君迟到了，并带有确认的语气。
森: すみません、課長。今朝 9時に 起きました。
森: すみません、課長かちょう。今朝けさ ９く時じに 起おきました。
森: 对不起，科长。我今天早晨9点才起床。
  • すみません: [感叹词/动词] 对不起，不好意思。用于道歉或引起注意。
  • 課長かちょう: [名词] 科长。职位名称，用于称呼。
  • 今朝けさ: [名词] 今天早上。
  • ９く時じ: [名词] 9点钟。
  • に: [格助詞 (かくじょし)]
    • 表示动作发生的具体时间点。
  • 起おきました: [动词]
    • 原型: 起おきる
    • 形式: ます形的过去式。
    • 含义: 起床了。
    • 结构: 起き + ました。
句意: 森君向科长道歉，并解释说今天早上9点才起床（暗示这是迟到的原因）。
