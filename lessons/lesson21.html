第21課 わたしは すき焼きを 食べた ことが あります

基本きほん

1. わたしは すき焼きを 食べた ことが あります。
1. わたしは すき焼やきを 食たべた ことが あります。
1. 我吃过日式牛肉火锅。
  • わたし: [代名词] 我。
  • は: [係助詞 (かかりじょし)]
    • 提示主题: 表明句子谈论的对象是“わたし”。“关于我...”。
    • 读作 わ (wa)。
  • すき焼やき: [名词] 日式牛肉火锅（寿喜烧）。
  • を: [格助詞 (かくじょし)]
    • 表示动作的对象: 表明“食べる (吃)”这个动作作用的对象是“すき焼き”。
  • 食たべた: [动词]
    • 原型: 食たべる (taberu) - 吃。属于二类动词（一段动词）。
    • 形式: 动词的「た形」(ta-form)，即简体过去式。
    • 构成: 食たべる + た -> 食たべた。
  • こと: [形式名词]
    • 将前面的动词短语名词化，指代“吃过（すき焼き）”这件事。
  • が: [格助詞 (かくじょし)]
    • 在这里用于连接形式名词 `こと` 和后续的 `あります`，构成固定句型 `〜た ことがある`。
  • あります: [动词]
    • 原型: ある (aru) - 有，存在（用于无生命物体或抽象事物）。
    • 形式: 动词的「ます形」(masu-form) 的现在肯定式，表示礼貌。
    • 句型: 〜た ことが あります 是一个固定句型，表示“有过...的经历”。
句意: 陈述自己有过吃日式牛肉火锅的经历。
2. 李さんは 会社が 終わった 後で、飲みに 行きます。
2. 李りさんは 会社かいしゃが 終おわった 後あとで、飲のみに 行いきます。
2. 李さん公司下班后，要去喝酒。
  • 李りさん: [名词] 李先生/女士。“さん”是常用的敬称后缀。
  • は: [係助詞 (かかりじょし)]
    • 提示主题: 表明句子谈论的对象是“李さん”。“关于李先生...”。
    • 读作 わ (wa)。
  • 会社かいしゃ: [名词] 公司。
  • が: [格助詞 (かくじょし)]
    • 表示主语: 在从句 `会社が終わった` 中，表示动作“終わる (结束)”的主语是“会社”。
  • 終おわった: [动词]
    • 原型: 終おわる (owaru) - 结束。属于一类动词（五段动词）。
    • 形式: 动词的「た形」(ta-form)，即简体过去式。
    • 构成: 終おわる + っ + た -> 終おわった (促音便)。
  • 後あとで: [名词 + 格助詞]
    • 结构: 动词た形 + 後あとで
    • 含义: 表示在一个动作发生之后再进行另一个动作。“...之后”。
    • 後あと 是名词，意为“之后”。で 是格助词，在这里表示时间点。
  • 飲のみに: [动词ます形去掉ます + 助詞]
    • 原型: 飲のむ (nomu) - 喝（这里引申为喝酒）。
    • 形式: 动词的「ます形」词干 (飲のみ) + に。
    • 含义: 表示去某地的目的。“（去）喝酒”。 に 在这里表示目的。
  • 行いきます: [动词]
    • 原型: 行いく (iku) - 去。
    • 形式: 动词的「ます形」(masu-form) 的现在肯定式，表示礼貌。
句意: 说明李先生在公司（的工作）结束后，要去喝酒。
3. もっと 野菜を 食べた ほうが いいですよ。
3. もっと 野菜やさいを 食たべた ほうが いいですよ。
3. 还是多吃点蔬菜的好啊。
  • もっと: [副词] 更加，更多。
  • 野菜やさい: [名词] 蔬菜。
  • を: [格助詞 (かくじょし)]
    • 表示动作的对象: 表明“食べる (吃)”这个动作作用的对象是“野菜”。
  • 食たべた: [动词]
    • 原型: 食たべる (taberu) - 吃。属于二类动词。
    • 形式: 动词的「た形」(ta-form)，简体过去式。用于构成 〜た ほうが いい 句型。
  • ほう: [名词] 方，方面。
  • が: [格助詞 (かくじょし)]
    • 在这里用于连接名词 `ほう` 和后续的 `いいです`，构成固定句型 `〜た ほうが いいです`。
  • いい: [形容词 (イ形容词)] 好的。
  • です: [助动词]
    • 丁寧体 (ていねいたい - Polite Style) 的判断助动词。接在形容词后面，使句子更礼貌。
  • よ: [終助詞 (しゅうじょし)]
    • 放在句末，起提醒、告知或轻微强调的作用。传达一种“你知道吗？”或“我告诉你”的语气。
  • 句型: 〜た ほうが いいです
    • 结构: 动词た形 + ほう が いいです。
    • 含义: 用于提出建议或忠告，表示“做...比较好”。
句意: 提出建议，认为（对方）多吃点蔬菜比较好。
4. 窓を 閉めましょうか。
4. 窓まどを 閉しめましょうか。
4. 我把窗户关上吧？
  • 窓まど: [名词] 窗户。
  • を: [格助詞 (かくじょし)]
    • 表示动作的对象: 表明“閉める (关)”这个动作作用的对象是“窓”。
  • 閉しめましょう: [动词]
    • 原型: 閉しめる (shimeru) - 关闭。属于二类动词。
    • 形式: 动词的「ましょう形」(mashou-form)，即意志形/劝诱形的礼貌体。
    • 构成: 閉しめる + ましょう -> 閉しめましょう。
    • 含义: 表示提议“让我们（一起）做...”或主动提出“我来做...吧”。
  • か: [終助詞 (しゅうじょし)]
    • 放在句末，表示疑问。
    • 与 ましょう 结合成 〜ましょうか，常用于主动提供帮助或征求对方意见，“我来...好吗？”或“我们一起...怎么样？”。在这里是主动提出关窗。
句意: 说话人主动提出要关窗户，并征求对方的意见。

基本きほん会話かいわ

A 甲: お寿司を 食べた ことがありますか。
A 甲: お寿司すしを 食たべた ことが ありますか。
A 甲: 你吃过寿司吗？
  • お寿司すし: [名词] 寿司。“お”是美化语前缀，表示礼貌或尊敬。
  • を: [格助詞] 表示动作“食べる”的对象是“お寿司”。
  • 食たべた: [动词] 食たべる 的た形 (简体过去式)。
  • こと: [形式名词] 将动词短语名词化，指代“吃过（寿司）”这件事。
  • が: [格助詞] 连接形式名词 `こと` 和 `あります`，构成 `〜た ことがある` 句型。
  • あります: [动词] ある 的ます形。与 `〜た こと が` 连用，表示过去的经历。
  • か: [終助詞] 表示疑问。
  • 句型: 〜た ことが ありますか 是询问对方是否有过做某事的经历。
句意: 甲向乙询问是否有吃过寿司的经历。
乙: いいえ、一度も ありません。天ぷらは 食べた ことが あります。
乙: いいえ、一度いちども ありません。天てんぷらは 食たべた ことが あります。
乙: 没有，一次也没有。倒是吃过天麸罗。
  • いいえ: [感叹词] 不，不是。用于否定回答。
  • 一度いちど: [副词/名词] 一次。
  • も: [係助詞]
    • 用法: 与表示疑问或不定范围的词（如 `一度`, `だれ`, `どこ` 等）连用，后接否定形式，表示全面否定。“（连）一次也（没）...”。
    • 含义: 一度いちども〜ない 表示“一次也（没）...”。
  • ありません: [动词]
    • 原型: ある (aru) - 有，存在。
    • 形式: 动词的「ます形」的否定式 (あります -> ありません)。
    • 含义: 在此句型 `〜た ことが ありません` 中，表示“没有过...的经历”。
  • 天てんぷら: [名词] 天妇罗。
  • は: [係助詞]
    • 提示主题/对比: 在这里有对比的意味，与前面否定的“寿司”相对，“（寿司没吃过，）至于天妇罗嘛...”
    • 读作 わ (wa)。
  • 食たべた: [动词] 食たべる 的た形。
  • こと が あります: [固定句型] 表示“有过...的经历”。
句意: 乙回答说一次寿司都没吃过，但是有过吃天妇罗的经历。
B 甲: この 資料を 見ても いいですか。
B 甲: この 資料しりょうを 見みても いいですか。
B 甲: 可以看看这个资料吗？
  • この: [连体词] 这个。修饰名词 `資料`，指代靠近说话人的事物。属于「こそあど言葉」的「こ」系列。
  • 資料しりょう: [名词] 资料。
  • を: [格助詞] 表示动作“見る”的对象是“資料”。
  • 見みて: [动词]
    • 原型: 見みる (miru) - 看。属于二类动词。
    • 形式: 动词的「て形」(te-form)。用于连接后续成分或构成句型。
    • 构成: 見みる + て -> 見みて。
  • も: [係助詞]
    • 句型: 〜て も いいです
    • 含义: 表示许可，“做...也可以”。
  • いい: [形容词 (イ形容词)] 好的。
  • です: [助动词] 使句子礼貌。
  • か: [終助詞] 表示疑问。
  • 句型: 〜て も いいですか 是请求许可的常用表达，“我可以做...吗？”。
句意: 甲向乙请求许可，询问是否可以看这份资料。
乙: はい。読んだ 後で、森さんに 渡して ください。
乙: はい。読よんだ 後あとで、森もりさんに 渡わたして ください。
乙: 可以，看完后请交给森先生。
  • はい: [感叹词] 是的，好的。用于肯定回答或应答。
  • 読よんだ: [动词]
    • 原型: 読よむ (yomu) - 读。属于一类动词。
    • 形式: 动词的「た形」(ta-form)，简体过去式。
    • 构成: 読よむ + ん + だ -> 読よんだ (拨音便)。
  • 後あとで: [名词 + 格助詞]
    • 结构: 动词た形 + 後あとで
    • 含义: 表示“读完之后”。
  • 森もりさん: [名词] 森先生/女士。
  • に: [格助詞]
    • 表示动作的对象/方向: 在这里表示“渡す (交给)”这个动作的对象是“森さん”。
  • 渡わたして: [动词]
    • 原型: 渡わたす (watasu) - 交给，递给。属于一类动词。
    • 形式: 动词的「て形」(te-form)。
    • 构成: 渡わたす + して -> 渡わたして。
  • ください: [补助动词]
    • 句型: 动词て形 + ください。
    • 含义: 表示请求或指示对方做某事，“请...”。是一种礼貌的请求。
句意: 乙同意甲看资料，并请求甲在读完之后将资料交给森先生。
C 甲: ホテルを 予約した ほうが いいですか。
C 甲: ホテルを 予約よやくした ほうが いいですか。
C 甲: 饭店还是先预约为好吧？
  • ホテル: [名词] 酒店，饭店。
  • を: [格助詞] 表示动作“予約する”的对象是“ホテル”。
  • 予約よやくした: [动词]
    • 原型: 予約よやくする (yoyaku suru) - 预约。属于三类动词（サ变动词）。
    • 形式: 动词的「た形」(ta-form)，简体过去式。
    • 构成: 予約よやくします + た -> 予約よやくした。
  • ほう が いいです: [固定句型]
    • 结构: 动词た形 + ほう が いいです。
    • 含义: 表示建议，“做...比较好”。
  • か: [終助詞] 表示疑问或确认。在这里是询问对方是否认为这样做比较好。
句意: 甲向乙询问，是不是先预约饭店比较好。
乙: そうですね。連休ですから、その ほうが いいですね。
乙: そうですね。連休れんきゅうですから、その ほうが いいですね。
乙: 是啊。因为是连休，先预约的好啊。
  • そうですね: [惯用语] 是啊，说的是呢。用于表示同意或附和对方的说法，并稍作思考。
  • 連休れんきゅう: [名词] 连休，连续假期。
  • です: [助动词] 判断助动词的礼貌体。
  • から: [接续助詞]
    • 用法: 接在名词+です、形容词、动词等活用语的终止形后面，表示原因或理由。“因为...，所以...”。
    • 含义: 連休れんきゅうですから 意为“因为是连休”。
  • その: [连体词] 那个。指代前面提到的“予約した（こと）”（预约这件事）。属于「こそあど言葉」的「そ」系列。
  • ほう が いいです: [固定句型] 表示“...那样比较好”。
  • ね: [終助詞]
    • 放在句末，表示确认、征求同意或感叹。这里表示同意对方的看法，并带有轻微的感叹或确认语气。
句意: 乙同意甲的看法，并说明理由是因为正值连休，所以先预约比较好。
D 甲: 何か 食べましょうか。
D 甲: 何なにか 食たべましょうか。
D 甲: 吃点什么吗？
  • 何なにか: [副词/不定代名词] 什么东西，某些东西。表示不特定的事物。
  • 食たべましょう: [动词]
    • 原型: 食たべる (taberu) - 吃。
    • 形式: 动词的「ましょう形」(mashou-form)，意志形/劝诱形的礼貌体。
    • 含义: 表示提议“我们吃...吧”。
  • か: [終助詞]
    • 与 ましょう 结合成 〜ましょうか，表示提议或邀请，“我们...怎么样？”。
句意: 甲向乙提议一起吃点什么东西。
乙: いいえ。寝る 前に 食べない ほうが いいですよ。
乙: いいえ。寝ねる 前まえに 食たべない ほうが いいですよ。
乙: 不，睡觉前还是不吃东西的好。
  • いいえ: [感叹词] 不。用于否定回答。
  • 寝ねる: [动词]
    • 原型: 寝ねる (neru) - 睡觉。属于二类动词。
    • 形式: 动词的原形（辞書形）。
  • 前まえに: [名词 + 格助詞]
    • 结构: 动词原形 + 前まえに。
    • 含义: 表示在一个动作发生之前。“在...之前”。
    • 前まえ 是名词，意为“前面，之前”。に 是格助词，表示时间点。
  • 食たべない: [动词]
    • 原型: 食たべる (taberu) - 吃。
    • 形式: 动词的「ない形」(nai-form)，简体否定式。
    • 构成: 食たべる + ない -> 食たべない。
  • ほう が いいです: [固定句型]
    • 结构: 动词ない形 + ほう が いいです。
    • 含义: 用于提出建议或忠告，表示“不做...比较好”。
  • よ: [終助詞] 起提醒或告知作用。
句意: 乙拒绝了甲的提议，并建议说睡觉前不吃东西比较好。

応用おうよう

(小野おの回国きこくの日ひが近ちかづきました。回国きこく前まえ、吉田よしだ課長かちょうは小しょう李りに来きて歌舞伎かぶき観賞かんしょうの票ひょうを買かいました。 夜場よるばの演出えんしゅつに間まに合あわせるため、小しょう李りと小野おのは前まえもって会社かいしゃを離はなれました。)
(4点前てんまえ、小しょう李りと小野おのは会社かいしゃを離はなれ、駅えきに向むかって歩あるいています)
小野: 少し 急ぎましょうか。遅れない ほうが いいですから。
小野おの: 少すこし 急いそぎましょうか。遅おくれない ほうが いいですから。
小野: 稍微快点吧？还是别迟到的好。
  • 少すこし: [副词] 少许，一点。
  • 急いそぎましょう: [动词]
    • 原型: 急いそぐ (isogu) - 着急，赶快。属于一类动词。
    • 形式: 动词的「ましょう形」(mashou-form)。
    • 构成: 急いそぐ + ぎましょう -> 急いそぎましょう。
    • 含义: 提议“我们快点吧”。
  • か: [終助詞] 与 `ましょう` 结合，表示提议或征求意见。
  • 遅おくれない: [动词]
    • 原型: 遅おくれる (okureru) - 迟到，晚。属于二类动词。
    • 形式: 动词的「ない形」(nai-form)，简体否定式。
    • 构成: 遅おくれる + ない -> 遅おくれない。
  • ほう が いいです: [固定句型]
    • 结构: 动词ない形 + ほう が いいです。
    • 含义: 表示建议，“不做...比较好”。这里省略了 `です`，变为简体 `ほうが いい`。
  • から: [接续助詞] 表示原因、理由。“因为...”。
句意: 小野提议稍微走快一点，并说明理由是因为不迟到比较好。
李: そう しましょう。ところで、小野さんは よく 歌舞伎を 見ますか。
李り: そう しましょう。ところで、小野おのさんは よく 歌舞伎かぶきを 見みますか。
李: 就这么办吧。对了，小野先生经常看歌舞伎吗？
  • そう: [副词] 那样地。指代小野前面提议的“急ぎましょう”（快点走）。
  • しましょう: [动词]
    • 原型: する (suru) - 做。属于三类动词。
    • 形式: 动词的「ましょう形」(mashou-form)。
    • 构成: します + ましょう -> しましょう。
    • 含义: そう しましょう 表示同意对方的提议，“就那样做吧”。
  • ところで: [接续词] （转换话题时用）话说，可是，对了。
  • 小野おのさん: [名词] 小野先生/女士。
  • は: [係助詞] 提示主题。读作 `わ (wa)`。
  • よく: [副词] 经常地，常常地。
  • 歌舞伎かぶき: [名词] 歌舞伎（日本传统戏剧）。
  • を: [格助詞] 表示动作“見る”的对象。
