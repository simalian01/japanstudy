第2課 これは本です

基本きほん

1. これは 本です。
1. これは 本ほんです。
1. 这是书。
  • これ: [指示代名詞 (しじだいめいし)]
    • 指代靠近说话人的事物。属于「こそあど言葉」中的「こ」系列 (近称)。
    • 含义: 这个。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“これ”。
    • 读音: わ (wa)。
    • 含义: 关于...；就...来说。
  • 本ほん: [名词]
    • 含义: 书。
  • です: [助动词]
    • 丁寧体 (ていねいたい - Polite Style) 的判断助动词。用于名词或形容动词词干后，表示肯定判断或陈述事实。
    • 含义: 是...。
    • 作用: 使句子表达礼貌。
句意: 礼貌地陈述靠近说话者的这个物体是“书”。
2. それは 何ですか。
2. それは 何なんですか。
2. 那是什么？
  • それ: [指示代名詞 (しじだいめいし)]
    • 指代靠近听话人的事物。属于「こそあど言葉」中的「そ」系列 (中称)。
    • 含义: 那个（离听者近）。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“それ”。
    • 读音: わ (wa)。
    • 含义: 关于...；就...来说。
  • 何なん: [代名詞 (疑问代词)]
    • 询问事物。
    • 当后面接「た」「だ」「な」行假名或助数词时，通常读作 なん。此处接续 です (だ行)，故读 なん。
    • 含义: 什么。
  • です: [助动词]
    • 丁寧体 (ていねいたい - Polite Style) 的判断助动词。
    • 含义: 是...。
    • 作用: 使句子表达礼貌。
  • か: [終助詞 (しゅうじょし)]
    • 接在句末，表示疑问。
    • 含义: ...吗？
    • 通常句尾语调上扬。
句意: 礼貌地询问靠近听话者的那个物体是“什么”。
3. あれは だれの 傘ですか。
3. あれは だれの 傘かさですか。
3. 那是谁的伞？
  • あれ: [指示代名詞 (しじだいめいし)]
    • 指代距离说话人和听话人都远的事物。属于「こそあど言葉」中的「あ」系列 (远称)。
    • 含义: 那个（离双方都远）。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“あれ”。
    • 读音: わ (wa)。
    • 含义: 关于...；就...来说。
  • だれ: [代名詞 (疑问代词)]
    • 询问人物。
    • 含义: 谁。
  • の: [格助詞 (かくじょし)]
    • 连接两个名词，表示所属关系（所有格）。
    • 结构: 名词1 + の + 名词2 (名词2属于名词1)。
    • 此处表示 “谁的”。
  • 傘かさ: [名词]
    • 含义: 伞。
  • です: [助动词]
    • 丁寧体 (ていねいたい - Polite Style) 的判断助动词。
    • 含义: 是...。
    • 作用: 使句子表达礼貌。
  • か: [終助詞 (しゅうじょし)]
    • 接在句末，表示疑问。
    • 含义: ...吗？
    • 通常句尾语调上扬。
句意: 礼貌地询问远离双方的“那个”是谁的“伞”。
4. この カメラは スミスさんのです。
4. この カメラは スミスさんのです。
4. 这个照相机是史密斯先生的。
  • この: [連体詞 (れんたいし)]
    • 指示限定词，用于修饰名词。指靠近说话人的事物。属于「こそあど言葉」中的「こ」系列 (近称)。
    • 结构: この + 名词。
    • 含义: 这个...。
    • 与 これ (代词) 不同，この 后面必须接名词。
  • カメラ: [名词]
    • 含义: 照相机 (来自英语 camera)。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“この カメラ”。
    • 读音: わ (wa)。
    • 含义: 关于...；就...来说。
  • スミス: [固有名词]
    • 含义: 史密斯 (姓氏)。
  • さん: [接尾詞 (せつびご)]
    • 接在人名或称谓后，表示尊敬或礼貌。
    • 含义: ...先生；...女士；...同学等 (通用敬称)。
  • の: [格助詞 (かくじょし)]
    • 此处用法特殊，接在表示人的名词（或代词）后面，省略了其后的名词，使其本身具有名词性质，表示“...的东西”。
    • 结构: (人/物) + の = ...的（东西）。
    • 含义: ...的。在此处表示 “史密斯先生的(东西/照相机)”。
  • です: [助动词]
    • 丁寧体 (ていねいたい - Polite Style) 的判断助动词。
    • 含义: 是...。
    • 作用: 使句子表达礼貌。
句意: 礼貌地陈述“这个照相机”是“史密斯先生的（照相机）”。

基本きほん会話かいわ

A 甲: これは テレビですか。
A 甲: これは テレビですか。
A 甲: 这是电视机吗？
  • これ: [指示代名詞] 这个 (近称)。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • テレビ: [名词] 电视机 (来自英语 television)。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，“吗？”。
句意: 礼貌地询问靠近说话者的这个物体是否是“电视机”。
乙: いいえ、それは テレビでは ありません。パソコンです。
乙: いいえ、それは テレビでは ありません。パソコンです。
乙: 不，那不是电视机。是计算机(个人电脑)。
  • いいえ: [感叹词]
    • 表示否定回答。
    • 含义: 不，不是。
  • それ: [指示代名詞] 那个 (指代甲刚才提到的 これ，即靠近乙的事物，中称)。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • テレビ: [名词] 电视机。
  • では ありません: [固定搭配]
    • 名词 + では ありません 是 です 的否定形式 (丁寧体)。
    • で 是助动词 だ 的连用形。は 是提示助词。ありません 是补助动词 ある 的否定敬体。
    • 口语中 では 常缩约为 じゃ (じゃ ありません)。
    • 含义: 不是...。
  • パソコン: [名词] 个人电脑 (来自 personal computer)。
  • です: [助动词] 礼貌判断助动词，“是”。
句意: 否定对方的猜测，指出那不是电视机，而是个人电脑。
B 甲: それは 何ですか。
B 甲: それは 何なんですか。
B 甲: 那是什么？
  • それ: [指示代名詞] 那个 (中称)。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • 何なん: [代名詞 (疑问代词)] 什么。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，“吗？”。
句意: 礼貌地询问靠近听话者的那个物体是“什么”。
乙: これは 日本語の 本です。
乙: これは 日本語にほんごの 本ほんです。
乙: 这是日语书。
  • これ: [指示代名詞] 这个 (回答甲的 それ，指靠近乙自己的事物，近称)。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • 日本語にほんご: [名词]
    • 含义: 日语。
  • の: [格助詞]
    • 连接两个名词，表示修饰关系或内容。
    • 结构: 名词1 + の + 名词2 (关于名词1的名词2)。
    • 此处表示 “日语的(书)”。
  • 本ほん: [名词] 书。
  • です: [助动词] 礼貌判断助动词，“是”。
句意: 回答对方的询问，说明这个物体是“日语书”。
C 甲: 森さんの かばんは どれですか。
C 甲: 森もりさんの かばんは どれですか。
C 甲: 森先生的包是哪个？
  • 森もり: [固有名词] 森 (姓氏)。
  • さん: [接尾詞] ...先生/女士 (敬称)。
  • の: [格助詞] 表示所属关系，“...的”。
  • かばん: [名词] 包。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • どれ: [指示代名詞 (疑问代词)]
    • 用于从三个或更多选项中进行选择性提问。属于「こそあど言葉」中的「ど」系列 (不定称)。
    • 含义: 哪个。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，“吗？”。
句意: 询问在一组包中，“哪一个”是“森先生的包”。
乙: あの かばんです。
乙: あの かばんです。
乙: 是那个。
  • あの: [連体詞]
    • 指示限定词，修饰名词。指远离说话人和听话人的事物。属于「こそあど言葉」中的「あ」系列 (远称)。
    • 结构: あの + 名词。
    • 含义: 那个...。
  • かばん: [名词] 包。
  • です: [助动词] 礼貌判断助动词，“是”。
句意: 指明“那个（远离双方的）包”就是森先生的包。
D 甲: その ノートは だれのですか。
D 甲: その ノートは だれのですか。
D 甲: 那个笔记本是谁的？
  • その: [連体詞]
    • 指示限定词，修饰名词。指靠近听话人的事物。属于「こそあど言葉」中的「そ」系列 (中称)。
    • 结构: その + 名词。
    • 含义: 那个...（靠近听者的）。
  • ノート: [名词] 笔记本 (来自英语 note)。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • だれ: [代名詞 (疑问代词)] 谁。
  • の: [格助詞]
    • 此处用法特殊，表示“...的东西”。
    • 结构: (人/疑问代词) + の = ...的（东西）。
    • 含义: ...的。だれの 意为 “谁的(东西/笔记本)”。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，“吗？”。
句意: 询问靠近听话者的“那个笔记本”是“谁的”。
乙: わたしのです。
乙: わたしのです。
乙: 是我的。
  • わたし: [代名詞]
    • 第一人称代词，男女通用，较礼貌。
    • 含义: 我。
  • の: [格助詞]
    • 此处用法特殊，表示“...的东西”。
    • 结构: (人/代词) + の = ...的（东西）。
    • 含义: ...的。わたしの 意为 “我的(东西/笔记本)”。
  • です: [助动词] 礼貌判断助动词，“是”。
句意: 回答对方的询问，说明那个笔记本是“我的”。

応用おうよう

(家族かぞくの写真しゃしん 小シャオ李リ計画けいかく先さきは渋谷しぶやの某ぼう賓館ひんかんに住すむ一段時間いちだんじかん、然しかる後のち再ふたたび公寓こうぐうに搬はんじる。此処この日ひ、小野おの来きて賓館ひんかんで小シャオ李リを見望けんぼうする。二人ふたりは大庁だいちょうで喫茶きっさ。)
(小李从包里取出记事本，从记事本里抽出一张照片)
小野: 李さん、それは 何ですか。
小野おの: 李リさん、それは 何なんですか。
小野: 李小姐，那是什么？
  • 小野おの: [固有名词] 小野 (姓氏)。
  • 李リ: [固有名词] 李 (姓氏)。
  • さん: [接尾詞] ...先生/女士 (敬称)。
  • それ: [指示代名詞] 那个 (指靠近李的照片，中称)。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • 何なん: [代名詞 (疑问代词)] 什么。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，“吗？”。
句意: 小野礼貌地询问李手中的那个东西是什么。
李: これですか。家族の 写真です。
李リ: これですか。家族かぞくの 写真しゃしんです。
李: 这个吗？这是(我)家人的照片。
  • 李リ: [固有名词] 李。
  • これ: [指示代名詞] 这个 (指自己手中的照片，近称)。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，此处表示确认，“...吗？”。
  • 家族かぞく: [名词]
    • 含义: 家人，家族。
  • の: [格助詞] 表示所属关系，“...的”。
  • 写真しゃしん: [名词]
    • 含义: 照片。
  • です: [助动词] 礼貌判断助动词，“是”。
句意: 李确认小野问的是这个，并说明这是家人的照片。
小野: (指着其中一个人) この 方は どなたですか。
小野おの: (指着其中一个人) この 方かたは どなたですか。
小野: (指着其中一个人)这位是谁呀？
  • 小野おの: [固有名词] 小野。
  • この: [連体詞] 这个... (修饰名词，近称)。
  • 方かた: [名词]
    • 指人的礼貌说法。
    • 含义: 位，这位，那位 (比 人ひと 更尊敬)。
  • は: [係助詞] 提示主题，读作 わ (wa)。
  • どなた: [代名詞 (疑问代词)]
    • 询问人物，是 だれ 的尊敬语。
    • 含义: 哪位，谁 (尊敬)。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，“吗？”。
句意: 小野指着照片中的某个人，用尊敬的语气询问“这位”是“哪位”。

李: 是我母亲。

  • の: [格助詞] 表示所属关系，“...的”。
  • 母はは: [名词]
    • 指自己的母亲。称呼别人的母亲时用 お母かあさん。
    • 含义: (我的)母亲。
  • です: [助动词] 礼貌判断助动词，“是”。
句意: 李回答说照片上的人是自己的母亲。

    • 含义: 多大年纪 (尊敬)。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，“吗？”。
句意: 小野用尊敬的语气询问李的母亲的年龄。

  • の: [格助詞] 表示材料或属性，“...的”。
  • ハンカチ: [名词]
    • 含义: 手绢 (来自英语 handkerchief)。
  • です: [助动词] 礼貌判断助动词，“是”。
  • か: [終助詞] 疑问助词，此处带有确认和感叹的语气。
句意: 小野惊喜地确认礼物是丝绸手绢。

    • 含义: 嗯，是的。

    • 含义: 汕头 (地名)。
  • の: [格助詞] 表示产地或来源，“...的”。

  • です: [助动词] 礼貌判断助动词，“是”。
  • 中国ちゅうごく: [固有名词]
    • 含义: 中国。
  • の: [格助詞] 表示所属关系或属性，“...的”。
  • 名産品めいさんひん: [名词]
    • 含义: 特产，名产。
  • です: [助动词] 礼貌判断助动词，“是”。

小野: どうも ありがとう ございます。
小野おの: どうも ありがとう ございます。
小野: 多谢了。
  • 小野おの: [固有名词] 小野。
  • どうも: [副词]
    • 可以加强语气，有“非常”、“实在”的意思。在此处与 ありがとう ございます 连用，表示非常感谢。单独使用也可表示感谢。
  • ありがとう: [形容詞/感叹词]
    • 原型是形容词 ありがたい (难得的，值得感谢的) 的连用形 ありがとう 演变而来。
    • 含义: 谢谢。
  • ございます: [補助動詞]
    • 动词 ござる 的ます形，是 ある 的丁宁语（非常礼貌的说法）。接在 ありがとう 后面，使其成为非常礼貌的感谢表达。
  • どうも ありがとう ございます: [固定表达]
    • 非常正式和礼貌的感谢语。
    • 含义: 非常感谢。
句意: 小野非常礼貌地向李表示感谢。
