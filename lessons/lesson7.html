第7課 李さんは 毎日 コーヒーを 飲みます

基本きほん

1. 李さんは 毎日 コーヒーを 飲みます。
1. 李りさんは 毎日まいにち コーヒーを 飲のみます。
1. 小李每天喝咖啡。
  • 李りさん: [固有名詞 + 接尾語] 对李先生/女士的称呼。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 毎日まいにち: [名詞] 每天。表示频率。
  • コーヒー: [名詞] 咖啡 (Coffee)。
  • を: [格助詞]
    • 表示他动词的直接宾语。这里指“喝”的对象是“咖啡”。
    • 读作 お (o)。
  • 飲のみます: [動詞]
    • 原型: 飲む (のむ) [五段活用动词]
    • 形式: ます形。表示习惯性动作或将来的动作，语气礼貌。
    • 含义: 喝。
句意: 陈述小李每天都喝咖啡这个习惯性动作，语气礼貌。
2. 李さんは 図書館で 勉強します。
2. 李りさんは 図書館としょかんで 勉強べんきょうします。
2. 小李在图书馆学习。
  • 李りさん: [固有名詞 + 接尾語] 对李先生/女士的称呼。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 図書館としょかん: [名詞] 图书馆。
  • で: [格助詞]
    • 表示动作发生的场所。“在图书馆”。
  • 勉強べんきょうします: [動詞]
    • 原型: 勉強する (べんきょうする) [サ行变格活用动词]
    • 形式: ます形。表示现在或将来的动作，或习惯性动作。
    • 含义: 学习。由名词「勉強」加上「する」构成。
句意: 陈述小李在图书馆这个场所进行学习的动作，语气礼貌。
3. わたしは 毎朝 パンか お粥を 食べます。
3. わたしは 毎朝まいあさ パンか お粥かゆを 食たべます。
3. 我每天早晨吃面包或(喝)稀饭。
  • わたし: [代名詞] 我。
  • は: [係助詞] 提示主题。读作 わ (wa)。
  • 毎朝まいあさ: [名詞] 每天早晨。
  • パン: [名詞] 面包 (源自葡萄牙语 pão)。
  • か: [副助詞]
    • 表示选择，“或者”。连接两个或多个名词，表示从中选择一个。
  • お粥かゆ: [名詞] 粥，稀饭。
    • お: [接頭語] 美化语，使词语更显礼貌或柔和。
  • を: [格助詞] 表示他动词的直接宾语，“吃”的对象是“面包或者粥”。读作 お (o)。
  • 食たべます: [動詞]
    • 原型: 食べる (たべる) [一段活用动词]
    • 形式: ます形。表示习惯性动作。
    • 含义: 吃。
句意: 陈述说话人每天早晨吃面包或者喝粥这个习惯，语气礼貌。
4. コーラと ケーキを ください。
4. コーラと ケーキを ください。
4. 请给我可乐和蛋糕。
  • コーラ: [名詞] 可乐 (Cola)。
  • と: [並列助詞]
    • 表示并列关系，“和”。用于列举事物。
  • ケーキ: [名詞] 蛋糕 (Cake)。
  • を: [格助詞] 表示要求的对象。读作 お (o)。
  • ください: [補助動詞・動詞]
    • 作为补助动词时，接在动词て形后，表示请求对方做某事。
    • 这里单独使用，前面接「名詞 + を」，表示请求给予该物品。“请给我...”。是「くれる」的尊敬语「くださる」的命令形，但作为固定句型表示客气的请求。
句意: 向对方请求给予可乐和蛋糕，是一种客气的请求，常用于购物或点餐。

    • 由疑问代词「何」加上表示全面的「も」构成，与否定形式呼应，表示完全否定。
  • 食たべませんでした: [動詞]
    • 原型: 食べる (たべる)
    • 形式: ます形的否定过去式 (ませんでした)。

句意: 回答今天早上什么东西都没有吃。

  • かしこまりました: [慣用句] 知道了，明白了（遵命）。
    • 是动词「かしこまる」（恭敬地接受命令或指示）的ます形过去式。
    • 是服务行业等场合接受顾客要求时的非常礼貌的应答语，比「わかりました」更郑重。
句意: 店员表示已经明白并接受了顾客的点餐要求。

基本

1. 小李每天喝咖啡。

2. 小李在图书馆学习。

3. 我每天早晨吃面包或(喝)稀饭。

4. 请给我可乐和蛋糕。

基本对话

A 甲: 小李，(你)早晨在家看报了吗?

乙: 没有，(我)没看。

B 甲: (你)早晨吃什么了?

乙: (我)什么都没吃。

C 甲: 吉田先生，你星期天干什么?

乙: 打网球或慢跑。

D 甲: 欢迎光临。

乙: 我买这个笔记本和铅笔。

応用

(路上，她们遇见提便利店袋子的吉田科长)

吉田: 小李，现在去吃午饭吗?

李: 是呀。和小野一起去。

小野: 科长您呢?

吉田: 我在便利店买了盒饭和茶水。

李: 您经常去便利店吗?

吉田: 不，平时一般去荞麦面馆，吃荞麦面条或者乌冬面条。

李: 是吗。

小野: 小李，那咱们今天去荞麦面馆怎么样?

李: 好吧。

小野: 那，科长，我们走了。

吉田: 去吧。
