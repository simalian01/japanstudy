第13課 机の上に 本が 3冊 あります

基本きほん

1. 机の上に 本が 3冊 あります。
1. 机つくえの上うえに 本ほんが 3冊さつ あります。
1. 桌子上有3本书。
  • 机つくえ: [名词] 桌子。
  • の: [格助詞 (かくじょし)]
    • 表示领属或所有关系。在这里连接名词“机”和表示位置的名词“上”，表示“桌子的(上面)”。
  • 上うえ: [名词] 上面，上方。
  • に: [格助詞 (かくじょし)]
    • 表示存在的场所。指明“本 (书)”存在的具体位置是“机の上 (桌子上)”。通常与存在动词「ある」、「いる」连用。
  • 本ほん: [名词] 书。
  • が: [格助詞 (かくじょし)]
    • 提示主语。在这里提示存在句的主语是“本 (书)”。当使用「あります / います」表示存在时，存在的对象通常用「が」来标记。
  • 3冊さつ: [数量词]
    • 3: [数词] 三。
    • 冊さつ: [量词] 用于数书本、杂志等册装物品的单位。
    • 表示数量，修饰“本”。
  • あります: [动词]
    • 原型: ある (aru)。
    • 形式: ます形 (丁寧体 - Polite Style)。
    • 含义: 表示无生命物体（或植物）的存在。“有”，“在”。
    • 结构: [场所] + に + [物] + が + あります。
句意: 描述桌子上面有三本书这个事实。
2. 李さんは 毎日 7時間 働きます。
2. 李りさんは 毎日まいにち 7時間じかん 働はたらきます。
2. 小李每天工作7小时。
  • 李りさん: [固有名词 + 后缀]
    • 李り: 姓氏“李”。
    • さん: [后缀] 加在人名后表示尊敬，相当于“先生”、“女士”等，比「様 (さま)」更常用。
  • は: [係助詞 (かかりじょし)]
    • 提示主题 (Topic Marker): 表明句子谈论的对象是“李さん”。“关于小李...”。
    • 读作 わ (wa)。
  • 毎日まいにち: [名词/副词] 每天。表示动作发生的频率。
  • 7時間じかん: [数量词]
    • 7: [数词] 七。
    • 時間じかん: [名词/量词] 时间；小时。这里作为量词，表示持续的时间长度。
    • 直接放在动词前，表示动作持续的时间。
  • 働はたらきます: [动词]
    • 原型: 働はたらく (hataraku)。
    • 形式: ます形 (丁寧体 - Polite Style)。
    • 含义: 工作，劳动。
    • 表示习惯性或重复性的动作。
句意: 说明小李每天的工作时长是7小时这一习惯性事实。
3. 李さんは 1週間に 2回 プールへ 行きます。
3. 李りさんは 1週間しゅうかんに 2回かい プールへ 行いきます。
3. 小李一周去两次游泳池(游泳)。
  • 李りさん: [固有名词 + 后缀] (说明同上句)
  • は: [係助詞 (かかりじょし)] (说明同上句)
  • 1週間しゅうかん: [名词]
    • 1: [数词] 一。
    • 週間しゅうかん: [名词] 星期，周。表示时间段。
  • に: [格助詞 (かくじょし)]
    • 表示频率的基准。指明“2回 (两次)”这个频率是在“1週間 (一周)”这个时间范围内发生的。“每...”。
    • 结构: [时间段] + に + [次数]
  • 2回かい: [数量词]
    • 2: [数词] 二。
    • 回かい: [量词] 次，回。用于数动作的次数。
  • プール: [名词] 游泳池 (Pool)。
  • へ: [格助詞 (かくじょし)]
    • 表示移动行为的方向或目的地。指明“行きます (去)”的目的地是“プール (游泳池)”。
    • 读作 え (e)。
    • 与「に」相比，更强调方向性。
  • 行いきます: [动词]
    • 原型: 行いく (iku)。
    • 形式: ます形 (丁寧体 - Polite Style)。
    • 含义: 去。
    • 表示移动行为，常与表示目的地的助词「へ」或「に」连用。
句意: 说明小李在一周这个时间段内去游泳池的频率是两次。
4. 午後 郵便局へ 荷物を 出しに 行きます。
4. 午後ごご 郵便局ゆうびんきょくへ 荷物にもつを 出だしに 行いきます。
4. 下午去邮局托运包裹。
  • 午後ごご: [名词] 下午。表示动作发生的时间。
  • 郵便局ゆうびんきょく: [名词] 邮局。
  • へ: [格助詞 (かくじょし)] (说明同上句) 表示去往的目的地是“邮局”。
  • 荷物にもつ: [名词] 行李，包裹。
  • を: [格助詞 (かくじょし)]
    • 表示动作的对象。在这里指明“出す (寄出)”这个动作的对象是“荷物 (包裹)”。
    • 读作 お (o)。
  • 出だしに: [动词连用形 + 助词]
    • 出だし: 动词「出だす (dasu, 寄出, 发出)」的ます形去掉「ます」后的形式 (也称为连用形)。
    • に: [格助詞 (かくじょし)] 表示目的。接在动词连用形后，与移动动词（如行く、来る、帰る）连用，表示去做某事。
    • 结构: [场所] + へ/に + [目的动作的连用形] + に + [移动动词]。
    • 含义: 为了做“出し(寄出)”这个动作。
  • 行いきます: [动词] (说明同上句) 去。
句意: 表述下午要去邮局做“寄包裹”这件事。

基本きほん会話かいわ

A 甲: すみません、はがきを 5枚 ください。
A 甲: すみません、はがきを 5枚まい ください。
A 甲: 对不起，请给我5张明信片。
  • すみません: [感叹词/动词]
    • 作为感叹词，用于打招呼、引起注意、道歉或表示感谢。这里是引起注意，相当于“Excuse me”。
    • 也可以看作动词「済すむ (sumu)」的否定ます形「済すみません」，原意是“（事情）没完，（给你添麻烦了）过意不去”。
  • はがき: [名词] 明信片。
  • を: [格助詞 (かくじょし)] (说明同上) 表示要求的对象是“はがき”。
  • 5枚まい: [数量词]
    • 5: [数词] 五。
    • 枚まい: [量词] 用于数扁平、薄的物品，如纸张、邮票、盘子、衬衫等。
  • ください: [动词/补助动词]
    • 作为动词「下くださる (kudasaru, 给我/授予我，是「くれる」的尊敬语)」的命令形（但语气是请求）。
    • 直接接在名词 + を 后面，表示“请给我（名词）”。
    • 也可以作为补助动词接在动词て形后，表示“请（为我）做某事”。
    • 是常用的请求表达。
句意: 甲向对方请求购买5张明信片。
乙: はい。5枚で 250円です。
乙: はい。5枚まいで 250円えんです。
乙: 好的。5张250日元。
  • はい: [感叹词] 是的，好的。表示肯定应答。
  • 5枚まい: [数量词] (说明同上) 5张。
  • で: [格助詞 (かくじょし)]
    • 表示总计或范围。在这里表示“5枚”合在一起的总价是“250円”。
    • 也可表示方式、手段、场所、原因理由等，此处表合计。
  • 250円えん: [数量词]
    • 250: [数词] 二百五十。
    • 円えん: [名词/量词] 日元 (日本货币单位)。
  • です: [助动词] (说明同基本句1) 丁寧体 (Polite Style) 的判断助动词，表示肯定判断。
句意: 乙回应甲的请求，并告知5张明信片的总价是250日元。
B 甲: よく 映画を 見ますか。
B 甲: よく 映画えいがを 見みますか。
B 甲: （你）经常看电影吗?
  • よく: [副词] 经常，常常。修饰动词“見ます”，表示动作发生的频率高。
  • 映画えいが: [名词] 电影。
  • を: [格助詞 (かくじょし)] (说明同上) 表示动作“見ます”的对象是“映画”。
  • 見みます: [动词]
    • 原型: 見みる (miru)。
    • 形式: ます形 (丁寧体 - Polite Style)。
    • 含义: 看。
  • か: [終助詞 (しゅうじょし)]
    • 接在句末，表示疑问。使陈述句变为疑问句。
    • 读升调。
句意: 甲询问乙是否经常看电影。
乙: ええ、1か月に 2回ぐらい 見ます。
乙: ええ、1か月げつに 2回かいぐらい 見みます。
乙: 是的。1个月看两次左右。
  • ええ: [感叹词] 嗯，是的。比「はい」稍微口语化、柔和一些的肯定应答。
  • 1か月げつ: [数量词]
    • 1: [数词] 一。
    • か月げつ: [量词] 个月。用于数月份的数量。注意读音是「かげつ」。月份名称用「がつ」(例: 1月 いちがつ)。
  • に: [格助詞 (かくじょし)] (说明同基本句3) 表示频率的基准。“每(一个月)”。
  • 2回かい: [数量词] (说明同基本句3) 两次。
  • ぐらい: [副助詞 (ふくじょし)]
    • 接在数量词后面，表示大概的数量、程度或时间。“大约”，“左右”。
    • 使表达不那么精确。
  • 見みます: [动词] (说明同上句) 看。
句意: 乙肯定地回答，并说明自己看电影的频率大约是一个月两次。
C 甲: 家から 会社まで どのぐらい かかりますか。
C 甲: 家うちから 会社かいしゃまで どのぐらい かかりますか。
C 甲: 从你家到公司需要多少时间?
  • 家うち: [名词] 家，房子。这里指对方的家。
  • から: [格助詞 (かくじょし)]
    • 表示起点（时间、空间）。“从...”。
  • 会社かいしゃ: [名词] 公司。
  • まで: [格助詞 (かくじょし)]
    • 表示终点（时间、空间）。“到...”。
    • 「から...まで」常连用，表示“从...到...”。
  • どのぐらい: [疑问词 + 副助词]
    • どの: [连体词] 哪一个，哪种程度。
    • ぐらい: [副助詞] (说明同上句) 大约。
    • 合在一起作为疑问词组，询问程度、数量、时间、距离等。“多少...”，“大约多少...”。这里询问所需时间。
  • かかります: [动词]
    • 原型: かかる (kakaru)。
    • 形式: ます形 (丁寧体 - Polite Style)。
    • 含义: 花费（时间、金钱）。
    • 与表示时间或金钱的词语连用。
  • か: [終助詞 (しゅうじょし)] (说明同上) 表示疑问。
句意: 甲询问乙从家到公司所需的时间大概是多少。
乙: 電車で 1時間ぐらい かかります。
乙: 電車でんしゃで 1時間じかんぐらい かかります。
乙: 乘电车需要1小时左右。
  • 電車でんしゃ: [名词] 电车。
  • で: [格助詞 (かくじょし)]
    • 表示交通方式或手段。“乘坐...”，“用...”。
  • 1時間じかん: [数量词] (说明同基本句2) 1小时。
  • ぐらい: [副助詞 (ふくじょし)] (说明同上句) 大约，左右。
  • かかります: [动词] (说明同上句) 花费（时间）。
句意: 乙回答说乘坐电车大约需要花费1小时。
D 甲: 昨日 何を しましたか。
D 甲: 昨日きのう 何なにを しましたか。
D 甲: （你）昨天干什么了?
  • 昨日きのう: [名词] 昨天。表示动作发生的时间。
  • 何なに: [疑问代名词] 什么。
  • を: [格助詞 (かくじょし)] (说明同上) 表示动作“しました”的对象是“何”。
  • しました: [动词]
    • 原型: する (suru)。
    • 形式: ます形的过去式 (丁寧体 - Polite Style)。する -> します -> しました。
    • 含义: 做，干。是应用范围很广的动词。
    • 与疑问词「何」连用，询问做了什么。
  • か: [終助詞 (しゅうじょし)] (说明同上) 表示疑问。
句意: 甲询问乙昨天做了什么事情。
乙: 新宿へ 映画を 見に 行きました。
乙: 新宿しんじゅくへ 映画えいがを 見みに 行いきました。
乙: 去新宿看电影了。
  • 新宿しんじゅく: [固有名词] 新宿 (地名)。
  • へ: [格助詞 (かくじょし)] (说明同基本句3) 表示移动的目的地。
  • 映画えいが: [名词] 电影。
  • を: [格助詞 (かくじょし)] (说明同上) 表示动作“見(る)”的对象是“映画”。
  • 見みに: [动词连用形 + 助词]
    • 見み: 动词「見みる (miru, 看)」的ます形去掉「ます」后的形式 (连用形)。
    • に: [格助詞 (かくじょし)] (说明同基本句4) 表示目的。接在动词连用形后，与移动动词连用，表示去做某事。
    • 含义: 为了看(电影)。
  • 行いきました: [动词]
    • 原型: 行いく (iku)。
    • 形式: ます形的过去式 (丁寧体 - Polite Style)。行く -> 行きます -> 行きました。
    • 含义: 去了。
句意: 乙回答说昨天去新宿的目的是看电影。

小野: これから 森さんと 近くの お店へ 飲みに 行きます。李さんも いっしょに どうですか。
小野おの: これから 森もりさんと 近ちかくの お店みせへ 飲のみに 行いきます。李りさんも いっしょに どうですか。
小野: 我等下要和森先生去附近的店里喝一杯。小李也一起来怎么样？
  • これから: [连语] 从现在起，接下来，等会儿。表示即将发生的动作。
  • 森もりさん: [固有名词 + 后缀] 森先生/女士。(说明同 李さん)
  • と: [格助詞 (かくじょし)]
    • 表示共同动作的对象。“和...一起”。
  • 近ちかく: [名词] 附近。
  • の: [格助詞 (かくじょし)] (说明同基本句1) 连接名词，表示“附近的”。
  • お店みせ: [名词]
    • お: [接头词] 美化语，加在某些名词前使其更礼貌或柔和。
    • 店みせ: [名词] 店铺。
  • へ: [格助詞 (かくじょし)] (说明同基本句3) 表示移动的目的地。
  • 飲のみに: [动词连用形 + 助词]
    • 飲のみ: 动词「飲のむ (nomu, 喝)」的ます形去掉「ます」后的形式 (连用形)。
    • に: [格助詞 (かくじょし)] (说明同基本句4) 表示目的。
    • 含义: 为了喝(酒)。
  • 行いきます: [动词] (说明同基本句3) 去。表示计划。
  • 李りさん: [固有名词 + 后缀] (说明同基本句2) 小李。
  • も: [係助詞 (かかりじょし)]
    • 表示类比、添加。“也...”。表示李さん也和前面提到的森さん以及说话人一样。
  • いっしょに: [副词] 一起。
  • どうですか: [疑问词 + 助动词 + 终助词]
    • どう: [副词/疑问词] 怎么样，如何。
    • です: [助动词] (说明同基本句1) 使表达礼貌。
    • か: [終助詞 (しゅうじょし)] (说明同对话B) 表示疑问。
    • 整个短语用于询问意见或提议，“...怎么样？”。
句意: 小野告诉小李自己等下要和森先生去附近的店里喝酒，并邀请小李一起去。
李: えっ、いいですか。お願いします。
李り: えっ、いいですか。お願ねがいします。
李: 啊，可以吗？那拜托了（麻烦您了/那就一起去吧）。
  • えっ: [感叹词] 表示惊讶或疑问。啊？欸？
  • いい: [形容词 (イ形容词)] 好的，可以的。
  • です: [助动词] (说明同基本句1) 使表达礼貌。
  • か: [終助詞 (しゅうじょし)] (说明同对话B) 表示疑问。
  • いいですか: 常见的表达，用于确认是否可以。“可以吗？”。
  • お願ねがいします: [固定表达]
    • お: [接头词] 美化语。
    • 願ねがい: 名词，「願ねがう (negau, 请求、期望)」的连用形名词化。
    • します: 动词「する (suru, 做)」的ます形。
    • 常用寒暄语，表示请求、拜托或接受好意。“拜托了”，“麻烦你了”。在这里表示接受邀请。
句意: 小李有些惊讶地确认是否真的可以一起去，然后欣然接受了邀请。
(居酒屋いざかやは 会社かいしゃ帰がえりの サラリーマンが 多おおくて、とても にぎやかです。)

李り: 森もりさんたちは この お店みせへ よく 来きますか。
李: 森先生你们经常来这家店吗？
